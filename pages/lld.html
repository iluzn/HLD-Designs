🎯 What Interviewers Grade On

1
Working Demo Must compile and run end-to-end with a main/demo class.
2
Separation of Concerns Multiple files/classes, each doing one thing well.
3
Strategy Pattern At least one strategy — pricing, eviction, assignment, scoring.
4
Extensibility Adding a new variant = one new class, no edits to existing code.
5
Edge Cases Null inputs, capacity limits, state machine violations.
6
Thread-Safety When shared mutable state exists, synchronize correctly.
