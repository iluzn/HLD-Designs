<h1>Subsets</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Backtracking</span><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Given an array <code>nums</code> of unique integers, return all possible subsets (the power set). The result may be in any order; this judge compares them order-independently.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3]
Output: [[],[1],[2],[3],[1,2],[1,3],[2,3],[1,2,3]]</pre>
<h2>Example 2</h2><pre>Input:  nums = [0]
Output: [[],[0]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10</li><li>All numbers are unique.</li></ul>
