<h1>Deepest Leaves Sum</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">BFS</span><span class="lc-tag">DFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, return the sum of values of its <strong>deepest</strong> leaves.</p>
<h2>Example 1</h2><pre>Input:  root = [1,2,3,4,5,null,6,7,null,null,null,null,8]
Output: 15</pre>
<h2>Example 2</h2><pre>Input:  root = [6,7,8,2,7,1,3,9,null,1,4,null,null,null,5]
Output: 19</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^4</li><li>1 &lt;= Node.val &lt;= 100</li></ul>
