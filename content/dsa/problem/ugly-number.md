<h1>Ugly Number</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span></div></details>
<p>An <strong>ugly number</strong> is a positive integer whose prime factors are limited to <code>2</code>, <code>3</code>, and <code>5</code>. Given an integer <code>n</code>, return <code>true</code> if <code>n</code> is ugly.</p>
<h2>Example 1</h2><pre>Input:  n = 6
Output: true</pre>
<h2>Example 2</h2><pre>Input:  n = 14
Output: false</pre>
<h2>Example 3</h2><pre>Input:  n = 1
Output: true</pre>
<h2>Constraints</h2><ul><li>-2^31 &lt;= n &lt;= 2^31 - 1</li></ul>
