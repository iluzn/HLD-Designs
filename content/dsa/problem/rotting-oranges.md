<h1>Rotting Oranges</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">BFS</span><span class="lc-tag">Matrix</span></div></details>
<p>In an <code>m x n</code> grid each cell is <code>0</code> (empty), <code>1</code> (fresh orange), or <code>2</code> (rotten). Every minute, a fresh orange adjacent (4-directionally) to a rotten one becomes rotten. Return the minutes until no fresh orange remains, or <code>-1</code> if impossible.</p>
<h2>Example 1</h2><pre>Input:  grid = [[2,1,1],[1,1,0],[0,1,1]]
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  grid = [[2,1,1],[0,1,1],[1,0,1]]
Output: -1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 10</li><li>grid[i][j] is 0, 1, or 2.</li></ul>
