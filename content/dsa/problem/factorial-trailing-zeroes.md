<p>Given an integer <code>n</code>, return the number of trailing zeroes in <code>n!</code> (n factorial).</p>
<h2>Example 1</h2><pre>Input:  n = 3
Output: 0
Explanation: 3! = 6, no trailing zero.</pre>
<h2>Example 2</h2><pre>Input:  n = 5
Output: 1
Explanation: 5! = 120.</pre>
<h2>Example 3</h2><pre>Input:  n = 0
Output: 0</pre>
