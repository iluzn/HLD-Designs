<h1>Find the Smallest Divisor Given a Threshold</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given an array <code>nums</code> and an integer <code>target</code> (the threshold), find the smallest positive divisor <code>d</code> such that the sum of <code>ceil(nums[i] / d)</code> over all elements is less than or equal to <code>target</code>. It is guaranteed that a valid divisor exists.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,5,9], target = 6
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  nums = [44,22,33,11,1], target = 5
Output: 44</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 5*10^4</li><li>1 &lt;= nums[i] &lt;= 10^6</li><li>nums.length &lt;= target &lt;= 10^6</li></ul>
