<h1>Linked List Cycle</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given the <code>head</code> of a linked list, return <code>true</code> if it has a cycle. On this judge the input is the node values plus <code>pos</code> — the index the tail’s <code>next</code> connects to, or <code>-1</code> for no cycle. <code>pos</code> is not passed to your function.</p>
<h2>Example 1</h2><pre>Input:  head = [3,2,0,-4], pos = 1
Output: true</pre>
<h2>Example 2</h2><pre>Input:  head = [1], pos = -1
Output: false</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^4</li><li>-10^5 &lt;= Node.val &lt;= 10^5</li><li>pos is -1 or a valid index.</li></ul>
