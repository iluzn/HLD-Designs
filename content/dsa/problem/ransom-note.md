<h1>Ransom Note</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">String</span><span class="lc-tag">Counting</span></div></details>
<p>Given two strings <code>s</code> (the ransom note) and <code>t</code> (the magazine), return <code>true</code> if <code>s</code> can be constructed using the letters of <code>t</code>. Each letter of <code>t</code> can be used at most once.</p>
<h2>Example 1</h2><pre>Input:  s = "aa", t = "aab"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "aa", t = "ab"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length, t.length &lt;= 10^5</li><li>s and t consist of lowercase English letters.</li></ul>
