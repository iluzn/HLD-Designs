<h1>Univalued Binary Tree</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span></div></details>
<p>A binary tree is <em>univalued</em> if every node has the same value. Given the <code>root</code>, return <code>true</code> if the tree is univalued, otherwise <code>false</code>.</p>
<h2>Example 1</h2><pre>Input:  root = [1,1,1,1,1,null,1]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  root = [2,2,2,5,2]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 100</li><li>0 &lt;= Node.val &lt;= 99</li></ul>
