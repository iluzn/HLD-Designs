<p>Given a non-empty array <code>nums</code> where every element appears twice except one, find that single one. Do it with O(n) time and O(1) extra space.</p>
<h2>Example 1</h2><pre>Input:  nums = [2,2,1]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  nums = [4,1,2,1,2]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 3*10^4</li><li>Each element appears twice except one.</li></ul>
