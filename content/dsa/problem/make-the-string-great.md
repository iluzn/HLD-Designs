<h1>Make The String Great</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Stack</span></div></details>
<p>A good string has no two adjacent characters where one is the lowercase and the other the uppercase of the same letter. Repeatedly remove such pairs from <code>s</code> and return the result.</p>
<h2>Example 1</h2><pre>Input:  s = "leEeetcode"
Output: "leetcode"</pre>
<h2>Example 2</h2><pre>Input:  s = "abBAcC"
Output: ""</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 100</li><li>s contains upper and lower case English letters.</li></ul>
