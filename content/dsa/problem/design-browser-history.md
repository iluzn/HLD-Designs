<h1>Design Browser History</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Linked List</span><span class="lc-tag">Stack</span><span class="lc-tag">Design</span></div></details>
<p>You have a browser with one tab starting on a homepage. Implement <code>BrowserHistory(homepage)</code>, <code>visit(url)</code> (visits <code>url</code> from the current page and clears all forward history), <code>back(steps)</code> (move back up to <code>steps</code> pages, return the current url), and <code>forward(steps)</code> (move forward up to <code>steps</code> pages, return the current url).</p>
<h2>Example 1</h2><pre>Input:  operations = ["BrowserHistory","visit","visit","visit","back","back","forward","visit","forward","back","back"], values = [["leetcode"],["google"],["facebook"],["youtube"],[1],[1],[1],["linkedin"],[2],[2],[7]]
Output: [null,null,null,null,facebook,google,facebook,null,linkedin,google,leetcode]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= homepage.length, url.length &lt;= 20</li><li>1 &lt;= steps &lt;= 100</li><li>At most 5000 calls.</li></ul>
