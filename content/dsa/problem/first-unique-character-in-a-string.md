<h1>First Unique Character in a String</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Queue</span></div></details>
<p>Given a string <code>s</code>, return the index of the first non-repeating character. If none exists, return <code>-1</code>.</p>
<h2>Example 1</h2><pre>Input:  s = "leetcode"
Output: 0</pre>
<h2>Example 2</h2><pre>Input:  s = "loveleetcode"
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  s = "aabb"
Output: -1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^5</li><li>s consists of lowercase English letters.</li></ul>
