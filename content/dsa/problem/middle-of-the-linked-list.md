<h1>Middle of the Linked List</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given the <code>head</code> of a singly linked list, return the list starting from the <strong>middle</strong> node. If there are two middle nodes, return the second one.</p>
<h2>Example 1</h2><pre>Input:  head = [1,2,3,4,5]
Output: [3,4,5]</pre>
<h2>Example 2</h2><pre>Input:  head = [1,2,3,4,5,6]
Output: [4,5,6]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 100</li><li>1 &lt;= Node.val &lt;= 100</li></ul>
