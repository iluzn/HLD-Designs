<h1>Single Number II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Every element in <code>nums</code> appears exactly three times except for one, which appears once. Return that single element. O(n) time and O(1) extra space.</p>
<h2>Example 1</h2><pre>Input:  nums = [2,2,3,2]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,1,0,1,0,1,99]
Output: 99</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 3*10^4</li><li>Each element appears 3x except one which appears once.</li></ul>
