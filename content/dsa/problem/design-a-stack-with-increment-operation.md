<p>Design a stack with a bulk-increment operation. Implement <code>CustomStack(maxSize)</code> and: <code>push(x)</code> (adds <code>x</code> only if the stack size is less than <code>maxSize</code>), <code>pop()</code> (removes and returns the top element, or <code>-1</code> if empty), and <code>increment(k, val)</code> (adds <code>val</code> to the bottom <code>k</code> elements — or all of them if the stack has fewer than <code>k</code>).</p>

Output: [null,null,null,2,null,null,null,null,null,103,202,201,-1]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= maxSize &lt;= 1000</li><li>1 &lt;= x, val &lt;= 100</li><li>0 &lt;= k &lt;= 1000</li><li>At most 1000 calls.</li></ul>
