<h1>Number of Good Pairs</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Math</span></div></details>
<p>Given an array <code>nums</code>, a pair <code>(i, j)</code> is good if <code>nums[i] == nums[j]</code> and <code>i &lt; j</code>. Return the number of good pairs.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,1,1,3]
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,1,1,1]
Output: 6</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 100</li><li>1 &lt;= nums[i] &lt;= 100</li></ul>
