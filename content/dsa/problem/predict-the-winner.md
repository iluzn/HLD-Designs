<h1>Predict the Winner</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Math</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Game Theory</span></div></details>
<p>Given a score array <code>nums</code>, two players take turns picking a number from either end; the picked value is added to that player&#39;s score. Player 1 moves first. Both play optimally. Return <code>true</code> if player 1 can end with a score greater than or equal to player 2.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,5,2]
Output: false</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,5,233,7]
Output: true</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 20</li><li>0 &lt;= nums[i] &lt;= 10^7</li></ul>
