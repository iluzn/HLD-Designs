<h1>Wildcard Matching</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Greedy</span></div></details>
<p>Given an input string <code>s</code> and a pattern <code>p</code>, return <code>true</code> if <code>p</code> matches the entire <code>s</code>. The pattern supports <code>'?'</code> (matches any single character) and <code>'*'</code> (matches any sequence of characters, including the empty sequence).</p>
<h2>Example 1</h2><pre>Input:  s = "aa", p = "a"
Output: false</pre>
<h2>Example 2</h2><pre>Input:  s = "aa", p = "*"
Output: true</pre>
<h2>Example 3</h2><pre>Input:  s = "cb", p = "?a"
Output: false</pre>
<h2>Constraints</h2><ul><li>0 &lt;= s.length, p.length &lt;= 16</li><li>s contains only lowercase letters.</li><li>p contains lowercase letters, '?' and '*'.</li></ul>
