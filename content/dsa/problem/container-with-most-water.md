<p>Given an integer array <code>height</code> of length n, find two lines that together with the x-axis form a container holding the most water. Return the maximum area.</p>
<h2>Example 1</h2><pre>Input:  height = [1,8,6,2,5,4,8,3,7]
Output: 49</pre>
<h2>Example 2</h2><pre>Input:  height = [1,1]
Output: 1</pre>
<h2>Constraints</h2><ul><li>2 &lt;= height.length &lt;= 10^5</li><li>0 &lt;= height[i] &lt;= 10^4</li></ul>
