<h1>Sum Root to Leaf Numbers</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span></div></details>
<p>Each root-to-leaf path in a binary tree represents a number formed by concatenating the node digits (each node value is a single digit <code>0-9</code>). Return the total sum of all root-to-leaf numbers.</p>
<h2>Example 1</h2><pre>Input:  root = [1,2,3]
Output: 25
Explanation: Paths 12 and 13; 12 + 13 = 25.</pre>
<h2>Example 2</h2><pre>Input:  root = [4,9,0,5,1]
Output: 1026</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 1000</li><li>0 &lt;= Node.val &lt;= 9</li></ul>
