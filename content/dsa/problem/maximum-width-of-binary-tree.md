<h1>Maximum Width of Binary Tree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">BFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, return the <strong>maximum width</strong>. The width of a level is the number of positions between its leftmost and rightmost non-null nodes (as if the level were a full binary tree), counting the null slots between them.</p>
<h2>Example 1</h2><pre>Input:  root = [1,3,2,5,3,null,9]
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  root = [1,3,2,5,null,null,9,6,null,7]
Output: 7</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 3000</li><li>-100 &lt;= Node.val &lt;= 100</li></ul>
