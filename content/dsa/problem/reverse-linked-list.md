<h1>Reverse Linked List</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Recursion</span></div></details>
<p>Given the <code>head</code> of a singly linked list, reverse the list and return the new head. Lists are shown as arrays of node values.</p>
<h2>Example 1</h2><pre>Input:  head = [1,2,3,4,5]
Output: [5,4,3,2,1]</pre>
<h2>Example 2</h2><pre>Input:  head = []
Output: []</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 5000</li><li>-5000 &lt;= Node.val &lt;= 5000</li></ul>
