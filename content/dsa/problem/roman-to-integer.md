<h1>Roman to Integer</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">Math</span><span class="lc-tag">String</span></div></details>
<p>Given a Roman numeral string <code>s</code>, convert it to an integer. Symbols: I=1, V=5, X=10, L=50, C=100, D=500, M=1000. A smaller symbol before a larger one is subtracted (e.g., IV = 4).</p>
<h2>Example 1</h2><pre>Input:  s = "III"
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  s = "LVIII"
Output: 58</pre>
<h2>Example 3</h2><pre>Input:  s = "MCMXCIV"
Output: 1994</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 15</li><li>s is a valid Roman numeral in the range [1, 3999].</li></ul>
