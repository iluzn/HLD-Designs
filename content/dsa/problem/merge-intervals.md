<h1>Merge Intervals</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sorting</span><span class="lc-tag">Intervals</span></div></details>
<p>Given an array of intervals (each <code>[start, end]</code>), merge all overlapping intervals and return the non-overlapping result sorted by start. On this judge intervals are shown separated by <code>|</code>.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,3],[2,6],[8,10],[15,18]]
Output: [[1,6],[8,10],[15,18]]</pre>
<h2>Example 2</h2><pre>Input:  grid = [[1,4],[4,5]]
Output: [[1,5]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= intervals.length &lt;= 10^4</li><li>intervals[i].length == 2</li></ul>
