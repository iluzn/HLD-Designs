<h1>Longest Valid Parentheses</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Stack</span></div></details>
<p>Given a string <code>s</code> containing just the characters <code>'('</code> and <code>')'</code>, return the length of the longest <strong>valid</strong> (well-formed) parentheses substring.</p>
<h2>Example 1</h2><pre>Input:  s = "(()"
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  s = ")()())"
Output: 4</pre>
<h2>Example 3</h2><pre>Input:  s = ""
Output: 0</pre>
<h2>Constraints</h2><ul><li>0 &lt;= s.length &lt;= 16</li><li>s[i] is '(' or ')'.</li></ul>
