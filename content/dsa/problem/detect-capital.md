<p>Given a word <code>s</code>, return <code>true</code> if its capital usage is correct: all uppercase, all lowercase, or only the first letter uppercase.</p>
<h2>Example 1</h2><pre>Input:  s = "USA"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "FlaG"
Output: false</pre>
<h2>Example 3</h2><pre>Input:  s = "Google"
Output: true</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 100</li><li>s consists of upper and lowercase English letters.</li></ul>
