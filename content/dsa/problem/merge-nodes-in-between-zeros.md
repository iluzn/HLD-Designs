<h1>Merge Nodes in Between Zeros</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Simulation</span></div></details>
<p>You are given the <code>head</code> of a linked list that begins and ends with <code>0</code>; between every pair of consecutive <code>0</code>s there is at least one node. Merge each block of nodes between two zeros into a single node whose value is their sum, then return the resulting list (which no longer contains any <code>0</code>).</p>
<h2>Example 1</h2><pre>Input:  head = [0,3,1,0,4,5,2,0]
Output: [4,11]</pre>
<h2>Example 2</h2><pre>Input:  head = [0,1,0,3,0,2,2,0]
Output: [1,3,4]</pre>
<h2>Constraints</h2><ul><li>3 &lt;= number of nodes &lt;= 2*10^5</li><li>0 &lt;= Node.val &lt;= 1000</li><li>The list starts and ends with 0.</li></ul>
