<h1>Reorder List</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Stack</span></div></details>
<p>Given the head of a singly linked list <code>L0 &rarr; L1 &rarr; ... &rarr; Ln-1 &rarr; Ln</code>, reorder it to <code>L0 &rarr; Ln &rarr; L1 &rarr; Ln-1 &rarr; L2 &rarr; ...</code>. Return the reordered list. (LeetCode mutates in place and returns void; here return the head.)</p>
<h2>Example 1</h2><pre>Input:  head = [1,2,3,4]
Output: [1,4,2,3]</pre>
<h2>Example 2</h2><pre>Input:  head = [1,2,3,4,5]
Output: [1,5,2,4,3]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 5*10^4</li><li>0 &lt;= Node.val &lt;= 1000</li></ul>
