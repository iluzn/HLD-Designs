<h1>Arranging Coins</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Binary Search</span></div></details>
<p>You have <code>n</code> coins and want to build a staircase where row <code>i</code> holds exactly <code>i</code> coins. The last row may be incomplete. Return the number of <strong>complete</strong> rows you can build.</p>
<h2>Example 1</h2><pre>Input:  n = 5
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  n = 8
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  n = 1
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2^31 - 1</li></ul>
