<h1>Minimum Moves to Equal Array Elements</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Math</span><span class="lc-tag">Greedy</span></div></details>
<p>Given an array <code>nums</code>, in one move you may increment <code>n - 1</code> elements by 1. Return the minimum number of moves to make all elements equal. Incrementing n-1 elements is equivalent to decrementing one, so the answer is the sum of each element minus the minimum.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,1,1]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>-10^9 &lt;= nums[i] &lt;= 10^9</li></ul>
