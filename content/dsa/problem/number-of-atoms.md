<h1>Number of Atoms</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">String</span><span class="lc-tag">Stack</span><span class="lc-tag">Sorting</span></div></details>
<p>Given a chemical <code>formula</code> (as string <code>s</code>), return the count of each atom. Output the element names in <strong>alphabetical order</strong>, each followed by its count if the count is greater than 1. Groups may be nested with parentheses and followed by a multiplier.</p>
<h2>Example 1</h2><pre>Input:  s = "H2O"
Output: "H2O"</pre>
<h2>Example 2</h2><pre>Input:  s = "Mg(OH)2"
Output: "H2MgO2"</pre>
<h2>Example 3</h2><pre>Input:  s = "K4(ON(SO3)2)3"
Output: "K4N3O21S6"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 16</li><li>s is a valid chemical formula of element tokens, digits, and parentheses.</li></ul>
