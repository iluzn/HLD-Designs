<h1>Split Array Largest Sum</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Greedy</span></div></details>
<p>Given an array <code>nums</code> and an integer <code>target</code> (= k), split <code>nums</code> into <code>target</code> non-empty contiguous subarrays so as to minimize the largest subarray sum. Return that minimized largest sum.</p>
<h2>Example 1</h2><pre>Input:  nums = [7,2,5,10,8], target = 2
Output: 18</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3,4,5], target = 2
Output: 9</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,4,4], target = 3
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 1000</li><li>0 &lt;= nums[i] &lt;= 10^6</li><li>1 &lt;= target &lt;= nums.length</li></ul>
