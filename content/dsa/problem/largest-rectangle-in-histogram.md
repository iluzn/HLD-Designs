<h1>Largest Rectangle in Histogram</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Stack</span><span class="lc-tag">Monotonic Stack</span></div></details>
<p>Given an array <code>heights</code> of bar heights (each width 1), return the area of the largest rectangle that fits in the histogram.</p>
<h2>Example 1</h2><pre>Input:  heights = [2,1,5,6,2,3]
Output: 10</pre>
<h2>Example 2</h2><pre>Input:  heights = [2,4]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= heights.length &lt;= 10^5</li><li>0 &lt;= heights[i] &lt;= 10^4</li></ul>
