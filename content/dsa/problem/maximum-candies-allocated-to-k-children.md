<h1>Maximum Candies Allocated to K Children</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given an array <code>nums</code> of candy piles and an integer <code>target</code> (= k children), split piles (never merge) so every child gets the same number of candies from a single sub-pile. Return the maximum number of candies each child can get; return <code>0</code> if it is impossible to give every child at least one candy.</p>
<h2>Example 1</h2><pre>Input:  nums = [5,8,6], target = 3
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  nums = [2,5], target = 11
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= 10^7</li><li>1 &lt;= target &lt;= 10^12</li></ul>
