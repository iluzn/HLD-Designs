<h1>Minimum Value to Get Positive Step by Step Sum</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Prefix Sum</span></div></details>
<p>Given an array <code>nums</code>, pick a positive start value. The step-by-step total (start plus a running prefix of nums) must never drop below 1. Return the minimum positive start value.</p>
<h2>Example 1</h2><pre>Input:  nums = [-3,2,-3,4,2]
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2]
Output: 1</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,-2,-3]
Output: 5</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 100</li><li>-100 &lt;= nums[i] &lt;= 100</li></ul>
