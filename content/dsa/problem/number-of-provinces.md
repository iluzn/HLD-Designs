<h1>Number of Provinces</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span><span class="lc-tag">Union Find</span></div></details>
<p>There are <code>n</code> cities. You are given an <code>n x n</code> matrix <code>grid</code> where <code>grid[i][j] = 1</code> means city <code>i</code> and city <code>j</code> are directly connected. A province is a group of directly or indirectly connected cities. Return the total number of provinces.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,1,0],[1,1,0],[0,0,1]]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  grid = [[1,0,0],[0,1,0],[0,0,1]]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 200</li><li>grid[i][j] is 1 or 0</li><li>grid[i][i] == 1</li><li>grid[i][j] == grid[j][i].</li></ul>
