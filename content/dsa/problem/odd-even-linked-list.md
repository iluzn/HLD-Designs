<h1>Odd Even Linked List</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span></div></details>
<p>Given the <code>head</code> of a singly linked list, group all the nodes at <strong>odd</strong> positions together followed by the nodes at <strong>even</strong> positions (1-indexed), preserving relative order within each group. Return the reordered list.</p>
<h2>Example 1</h2><pre>Input:  head = [1,2,3,4,5]
Output: [1,3,5,2,4]</pre>
<h2>Example 2</h2><pre>Input:  head = [2,1,3,5,6,4,7]
Output: [2,3,6,7,1,5,4]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^4</li><li>-10^6 &lt;= Node.val &lt;= 10^6</li></ul>
