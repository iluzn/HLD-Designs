<h1>Isomorphic Strings</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">String</span></div></details>
<p>Given two strings <code>s</code> and <code>t</code>, return <code>true</code> if they are isomorphic — the characters in <code>s</code> can be replaced to get <code>t</code> with a consistent one-to-one mapping (no two characters map to the same character).</p>
<h2>Example 1</h2><pre>Input:  s = "egg", t = "add"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "foo", t = "bar"
Output: false</pre>
<h2>Example 3</h2><pre>Input:  s = "paper", t = "title"
Output: true</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 5*10^4</li><li>t.length == s.length</li><li>s and t consist of any valid ASCII characters.</li></ul>
