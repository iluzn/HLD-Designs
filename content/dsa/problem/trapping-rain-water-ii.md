<h1>Trapping Rain Water II</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">BFS</span><span class="lc-tag">Heap</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>m x n</code> matrix of non-negative integers representing the height of each cell, return the volume of water that can be trapped after raining.</p>
<h2>Example 1</h2><pre>Input:  heightMap = [[1,4,3,1,3,2],[3,2,1,3,2,4],[2,3,3,2,3,1]]
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  heightMap = [[3,3,3,3,3],[3,2,2,2,3],[3,2,1,2,3],[3,2,2,2,3],[3,3,3,3,3]]
Output: 10</pre>
<h2>Constraints</h2><ul><li>m == heightMap.length</li><li>n == heightMap[i].length</li><li>1 &lt;= m, n &lt;= 200</li><li>0 &lt;= heightMap[i][j] &lt;= 2*10^4.</li></ul>
