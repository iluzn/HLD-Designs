<h1>Non-overlapping Intervals</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span><span class="lc-tag">Sorting</span><span class="lc-tag">Intervals</span></div></details>
<p>Given an array of intervals (each <code>[start, end]</code>), return the minimum number of intervals you must remove so the rest are non-overlapping.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,2],[2,3],[3,4],[1,3]]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  grid = [[1,2],[1,2],[1,2]]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= intervals.length &lt;= 10^5</li><li>intervals[i].length == 2</li></ul>
