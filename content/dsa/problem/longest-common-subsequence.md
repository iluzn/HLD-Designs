<h1>Longest Common Subsequence</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given two strings <code>text1</code> and <code>text2</code>, return the length of their longest common subsequence (characters in order, not necessarily contiguous). Return 0 if none.</p>
<h2>Example 1</h2><pre>Input:  text1 = "abcde", text2 = "ace"
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  text1 = "abc", text2 = "def"
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= text1.length, text2.length &lt;= 1000</li><li>Inputs consist of lowercase English letters.</li></ul>
