<h1>Longest ZigZag Path in a Binary Tree</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>A ZigZag path alternates direction (left, right, left, ...) at each step. Its length is the number of <strong>edges</strong> visited. Given the <code>root</code> of a binary tree, return the length of the longest ZigZag path contained in it.</p>
<h2>Example 1</h2><pre>Input:  root = [1,null,1,1,1,null,null,1,1,null,1,null,null,null,1,null,1]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  root = [1,1,1,null,1,null,null,1,1,null,1]
Output: 4</pre>
<h2>Example 3</h2><pre>Input:  root = [1]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 5*10^4</li><li>1 &lt;= Node.val &lt;= 100.</li></ul>
