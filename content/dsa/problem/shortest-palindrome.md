<h1>Shortest Palindrome</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Rolling Hash</span><span class="lc-tag">String Matching</span></div></details>
<p>You are given a string <code>s</code>. You may convert it to a palindrome by adding characters in front of it. Return the shortest palindrome you can find by performing this transformation.</p>
<h2>Example 1</h2><pre>Input:  s = "aacecaaa"
Output: "aaacecaaa"</pre>
<h2>Example 2</h2><pre>Input:  s = "abcd"
Output: "dcbabcd"</pre>
<h2>Example 3</h2><pre>Input:  s = ""
Output: ""</pre>
<h2>Constraints</h2><ul><li>0 &lt;= s.length &lt;= 16</li><li>s consists of lowercase English letters only.</li></ul>
