<h1>Invert Binary Tree</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, invert it (swap every left and right child) and return the root. Output is the inverted tree in level order.</p>
<h2>Example 1</h2><pre>Input:  root = [4,2,7,1,3,6,9]
Output: [4,7,2,9,6,3,1]</pre>
<h2>Example 2</h2><pre>Input:  root = []
Output: []</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 100</li><li>-100 &lt;= Node.val &lt;= 100</li></ul>
