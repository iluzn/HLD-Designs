<h1>Add Strings</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">String</span><span class="lc-tag">Simulation</span></div></details>
<p>Given two non-negative integers <code>num1</code> and <code>num2</code> represented as strings, return their sum as a string. You must not convert the inputs to integers directly or use a big-integer library.</p>
<h2>Example 1</h2><pre>Input:  num1 = "11", num2 = "123"
Output: "134"</pre>
<h2>Example 2</h2><pre>Input:  num1 = "456", num2 = "77"
Output: "533"</pre>
<h2>Example 3</h2><pre>Input:  num1 = "0", num2 = "0"
Output: "0"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= num1.length, num2.length &lt;= 10^4</li><li>num1 and num2 consist of only digits.</li><li>num1 and num2 do not contain any leading zero except the number 0 itself.</li></ul>
