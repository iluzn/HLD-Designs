<h1>Integer Replacement</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Bit Manipulation</span><span class="lc-tag">Greedy</span></div></details>
<p>Given a positive integer <code>n</code>, in one operation you may replace <code>n</code> by <code>n / 2</code> if it is even, or by <code>n + 1</code> or <code>n - 1</code> if it is odd. Return the minimum number of operations needed for <code>n</code> to become <code>1</code>.</p>
<h2>Example 1</h2><pre>Input:  n = 8
Output: 3
Explanation: 8 -> 4 -> 2 -> 1.</pre>
<h2>Example 2</h2><pre>Input:  n = 7
Output: 4
Explanation: 7 -> 8 -> 4 -> 2 -> 1.</pre>
<h2>Example 3</h2><pre>Input:  n = 1
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2^31 - 1</li></ul>
