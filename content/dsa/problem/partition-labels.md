<h1>Partition Labels</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Greedy</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Hash Table</span></div></details>
<p>You are given a string <code>s</code>. Partition it into as many parts as possible so that each letter appears in at most one part. Return a list of the sizes of these parts, in order.</p>
<h2>Example 1</h2><pre>Input:  s = "ababcbacadefegdehijhklij"
Output: [9,7,8]</pre>
<h2>Example 2</h2><pre>Input:  s = "eccbbbbdec"
Output: [10]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 500</li><li>s consists of lowercase English letters.</li></ul>
