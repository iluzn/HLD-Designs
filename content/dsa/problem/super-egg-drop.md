<h1>Super Egg Drop</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>You are given <code>a</code> identical eggs and access to a building with <code>b</code> floors. An egg breaks if dropped from a floor at or above some unknown threshold <code>f</code> (<code>0 &lt;= f &lt;= b</code>) and survives below it. Return the <strong>minimum number of moves</strong> needed to determine <code>f</code> with certainty in the worst case.</p>
<h2>Example 1</h2><pre>Input:  a = 1, b = 2
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  a = 2, b = 6
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  a = 3, b = 14
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= a &lt;= 100</li><li>1 &lt;= b &lt;= 10^4</li></ul>
