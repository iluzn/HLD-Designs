<h1>Find K-th Smallest Pair Distance</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Sorting</span></div></details>
<p>The distance of a pair <code>(a, b)</code> is <code>|a - b|</code>. Given an integer array <code>nums</code> and an integer <code>k</code>, return the <code>k</code>-th smallest distance among all <code>n * (n - 1) / 2</code> pairs (1-indexed).</p>
<h2>Example 1</h2><pre>Input:  nums = [1,3,1], k = 1
Output: 0</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,1,1], k = 2
Output: 0</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,6,1], k = 3
Output: 5</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 10^4</li><li>0 &lt;= nums[i] &lt;= 10^6</li><li>1 &lt;= k &lt;= n * (n - 1) / 2</li></ul>
