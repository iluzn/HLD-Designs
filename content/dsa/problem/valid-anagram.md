<h1>Valid Anagram</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Sorting</span></div></details>
<p>Given two strings <code>s</code> and <code>t</code>, return <code>true</code> if <code>t</code> is an anagram of <code>s</code> (same letters, same counts).</p>
<h2>Example 1</h2><pre>Input:  s = "anagram", t = "nagaram"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "rat", t = "car"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length, t.length &lt;= 5*10^4</li><li>s and t consist of lowercase English letters.</li></ul>
