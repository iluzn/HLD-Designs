<h1>3Sum</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an integer array <code>nums</code>, return all unique triplets <code>[a, b, c]</code> that sum to 0. The result may be in any order and must not contain duplicate triplets.</p>
<h2>Example 1</h2><pre>Input:  nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,0,0]
Output: [[0,0,0]]</pre>
<h2>Constraints</h2><ul><li>3 &lt;= nums.length &lt;= 3000</li><li>-10^5 &lt;= nums[i] &lt;= 10^5</li></ul>
