<h1>Burst Balloons</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Interval DP</span></div></details>
<p>You have <code>n</code> balloons with numbers in <code>nums</code>. Bursting balloon <code>i</code> gives <code>nums[i-1]*nums[i]*nums[i+1]</code> coins (treat out-of-range as 1). Return the maximum coins from bursting all balloons.</p>
<h2>Example 1</h2><pre>Input:  nums = [3,1,5,8]
Output: 167</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,5]
Output: 10</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 300</li><li>0 &lt;= nums[i] &lt;= 100</li></ul>
