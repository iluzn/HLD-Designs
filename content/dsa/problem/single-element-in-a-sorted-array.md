<h1>Single Element in a Sorted Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given a sorted array <code>nums</code> where every element appears exactly twice except for one element that appears exactly once, return the single element. Your solution must run in O(log n) time and O(1) space.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,1,2,3,3,4,4,8,8]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [3,3,7,7,10,11,11]
Output: 10</pre>
<h2>Example 3</h2><pre>Input:  nums = [1]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>0 &lt;= nums[i] &lt;= 10^5</li><li>Every element appears twice except one which appears once.</li></ul>
