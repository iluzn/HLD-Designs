<h1>Is Subsequence</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given two strings <code>s</code> and <code>t</code>, return <code>true</code> if <code>s</code> is a subsequence of <code>t</code> (formed by deleting some characters of <code>t</code> without changing the order of the rest).</p>
<h2>Example 1</h2><pre>Input:  s = "abc", t = "ahbgdc"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "axc", t = "ahbgdc"
Output: false</pre>
<h2>Constraints</h2><ul><li>0 &lt;= s.length &lt;= 100</li><li>0 &lt;= t.length &lt;= 10^4</li></ul>
