<h1>Binary Tree Right Side View</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">BFS</span><span class="lc-tag">DFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, imagine standing on the right side. Return the values of the nodes you can see, ordered top to bottom.</p>
<h2>Example 1</h2><pre>Input:  root = [1,2,3,null,5,null,4]
Output: [1,3,4]</pre>
<h2>Example 2</h2><pre>Input:  root = [1,null,3]
Output: [1,3]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 100</li><li>-100 &lt;= Node.val &lt;= 100</li></ul>
