<h1>Single Number III</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Given an array <code>nums</code> in which exactly two elements appear only once and all the others appear exactly twice, return the two single numbers <strong>sorted in ascending order</strong>. Use O(n) time and O(1) extra space.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,1,3,2,5]
Output: [3,5]</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,1]
Output: [0,1]</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 3*10^4</li><li>Exactly two elements appear once; the rest appear twice.</li></ul>
