<h1>Task Scheduler</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span><span class="lc-tag">Heap</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given a string <code>tasks</code> of uppercase-letter task types and a cooldown <code>n</code> (passed as the second value), each task takes one unit and identical tasks must be at least <code>n</code> units apart. Return the minimum number of units to finish all tasks.</p>
<h2>Example 1</h2><pre>Input:  tasks = "AAABBB", n = 2
Output: 8</pre>
<h2>Example 2</h2><pre>Input:  tasks = "AAABBB", n = 0
Output: 6</pre>
<h2>Constraints</h2><ul><li>1 &lt;= tasks.length &lt;= 10^4</li><li>tasks are uppercase English letters.</li><li>0 &lt;= n &lt;= 100</li></ul>
