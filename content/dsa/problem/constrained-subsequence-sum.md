<h1>Constrained Subsequence Sum</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Queue</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Heap</span><span class="lc-tag">Monotonic Queue</span></div></details>
<p>Given an integer array <code>nums</code> and an integer <code>target</code> (= k), return the maximum sum of a <strong>non-empty subsequence</strong> such that for every two consecutive chosen elements at indices <code>i &lt; j</code>, the condition <code>j - i &lt;= target</code> holds.</p>
<h2>Example 1</h2><pre>Input:  nums = [10,2,-10,5,20], target = 2
Output: 37</pre>
<h2>Example 2</h2><pre>Input:  nums = [-1,-2,-3], target = 1
Output: -1</pre>
<h2>Example 3</h2><pre>Input:  nums = [10,-2,-10,-5,20], target = 2
Output: 23</pre>
<h2>Constraints</h2><ul><li>1 &lt;= target &lt;= nums.length &lt;= 10^5</li><li>-10^4 &lt;= nums[i] &lt;= 10^4</li></ul>
