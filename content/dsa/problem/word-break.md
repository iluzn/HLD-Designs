<h1>Word Break</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given a string <code>s</code> and a dictionary of strings <code>wordDict</code>, return <code>true</code> if <code>s</code> can be segmented into a space-separated sequence of one or more dictionary words. A word may be reused.</p>
<h2>Example 1</h2><pre>Input:  s = "leetcode", wordDict = ["leet","code"]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "catsandog", wordDict = ["cats","dog","sand","and","cat"]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 300</li><li>1 &lt;= wordDict.length &lt;= 1000</li><li>All dictionary words are unique.</li></ul>
