<h1>Letter Combinations of a Phone Number</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Backtracking</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given a string of <code>digits</code> from 2-9, return all letter combinations the number could spell (phone keypad mapping). Return an empty result for an empty input. Order does not matter.</p>
<h2>Example 1</h2><pre>Input:  digits = "23"
Output: ["ad","ae","af","bd","be","bf","cd","ce","cf"]</pre>
<h2>Example 2</h2><pre>Input:  digits = ""
Output: []</pre>
<h2>Constraints</h2><ul><li>0 &lt;= digits.length &lt;= 4</li><li>digits[i] is in the range 2-9.</li></ul>
