<p>Given the <code>root</code> of a binary tree, return its <strong>inorder</strong> traversal (left, node, right) as a list of values.</p>
<h2>Example 1</h2><pre>Input:  root = [1,null,2,3]
Output: [1,3,2]</pre>
<h2>Example 2</h2><pre>Input:  root = []
Output: []</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 100</li><li>-100 &lt;= Node.val &lt;= 100</li></ul>
