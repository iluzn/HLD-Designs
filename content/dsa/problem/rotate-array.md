<h1>Rotate Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given an array <code>nums</code>, rotate it to the right by <code>k</code> steps (<code>k</code> passed as <code>target</code>) and return the rotated array.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,4,5,6,7], k = 3
Output: [5,6,7,1,2,3,4]</pre>
<h2>Example 2</h2><pre>Input:  nums = [-1,-100,3,99], k = 2
Output: [3,99,-1,-100]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>0 &lt;= k &lt;= 10^5</li></ul>
