<h1>Target Sum</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Backtracking</span></div></details>
<p>Given an array <code>nums</code> and an integer target <code>k</code>, assign a <code>+</code> or <code>-</code> to each number and count the assignments whose sum equals <code>k</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,1,1,1,1], k = 3
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  nums = [1], k = 1
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 20</li><li>0 &lt;= nums[i] &lt;= 1000</li><li>-1000 &lt;= k &lt;= 1000</li></ul>
