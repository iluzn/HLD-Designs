<h1>H-Index</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sorting</span><span class="lc-tag">Counting Sort</span></div></details>
<p>Given an array <code>citations</code> where <code>citations[i]</code> is the number of citations for the i-th paper, return the researcher's <strong>h-index</strong>: the maximum h such that h papers have at least h citations each.</p>
<h2>Example 1</h2><pre>Input:  citations = [3,0,6,1,5]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  citations = [1,3,1]
Output: 1</pre>
<h2>Constraints</h2><ul><li>n == citations.length</li><li>1 &lt;= n &lt;= 5000</li><li>0 &lt;= citations[i] &lt;= 1000</li></ul>
