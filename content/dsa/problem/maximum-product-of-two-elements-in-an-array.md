<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code>, choose two different indices i and j and return the maximum value of <code>(nums[i]-1)*(nums[j]-1)</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [3,4,5,2]
Output: 12</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,5,4,5]
Output: 16</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 500</li><li>1 &lt;= nums[i] &lt;= 10^3</li></ul>
