<p>Given an input string <code>s</code>, reverse the order of the words. Words are separated by one or more spaces; the result must have single spaces between words and no leading or trailing spaces.</p>
<h2>Example 1</h2><pre>Input:  s = "the sky is blue"
Output: "blue is sky the"</pre>
<h2>Example 2</h2><pre>Input:  s = "  hello world  "
Output: "world hello"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^4</li></ul>
