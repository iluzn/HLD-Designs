<h1>Minimum Height Trees</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">BFS</span><span class="lc-tag">Topological Sort</span></div></details>
<p>Given a tree of <code>n</code> nodes labeled from <code>0</code> to <code>n-1</code> with <code>n-1</code> edges, return the <strong>number</strong> of nodes that could be the root of a Minimum Height Tree (MHT). A MHT is one where the longest path from root to any leaf is minimized.</p>
<h2>Example 1</h2><pre>Input:  n = 4, edges = [[1,0],[1,2],[1,3]]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  n = 6, edges = [[3,0],[3,1],[3,2],[3,4],[5,4]]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2*10^4</li><li>edges.length == n - 1</li><li>The graph is a valid tree.</li></ul>
