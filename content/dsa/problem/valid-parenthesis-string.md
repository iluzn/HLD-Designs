<h1>Valid Parenthesis String</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Greedy</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Stack</span></div></details>
<p>Given a string <code>s</code> containing only <code>(</code>, <code>)</code> and <code>*</code>, return <code>true</code> if it can be made a valid parentheses string. A <code>*</code> can be treated as <code>(</code>, <code>)</code>, or an empty string.</p>
<h2>Example 1</h2><pre>Input:  s = "()"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "(*)"
Output: true</pre>
<h2>Example 3</h2><pre>Input:  s = "(*))"
Output: true</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 100</li><li>s consists of '(', ')' and '*'.</li></ul>
