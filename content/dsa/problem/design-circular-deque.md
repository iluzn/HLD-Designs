<h1>Design Circular Deque</h1>

<p>Design a circular double-ended queue. Implement <code>MyCircularDeque(k)</code> and: <code>insertFront(value)</code>, <code>insertLast(value)</code>, <code>deleteFront()</code>, <code>deleteLast()</code> (each returns true on success), <code>getFront()</code> and <code>getRear()</code> (return the value or <code>-1</code> if empty), <code>isEmpty()</code>, and <code>isFull()</code>.</p>
<h2>Example 1</h2><pre>Input:  operations = ["MyCircularDeque","insertLast","insertLast","insertFront","insertFront","getRear","isFull","deleteLast","insertFront","getFront"], values = [[3],[1],[2],[3],[4],[],[],[],[4],[]]
Output: [null,true,true,true,false,2,true,true,true,4]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= k &lt;= 1000</li><li>0 &lt;= value &lt;= 1000</li><li>At most 2000 calls.</li></ul>
