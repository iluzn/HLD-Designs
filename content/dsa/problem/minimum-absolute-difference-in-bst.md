<h1>Minimum Absolute Difference in BST</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BST</span></div></details>
<p>Given the <code>root</code> of a Binary Search Tree, return the minimum absolute difference between the values of any two <strong>different</strong> nodes.</p>
<h2>Example 1</h2><pre>Input:  root = [4,2,6,1,3]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  root = [1,0,48,null,null,12,49]
Output: 1</pre>
<h2>Constraints</h2><ul><li>2 &lt;= number of nodes &lt;= 10^4</li><li>0 &lt;= Node.val &lt;= 10^5</li><li>The tree is a valid BST.</li></ul>
