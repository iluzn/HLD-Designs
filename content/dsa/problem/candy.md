<h1>Candy</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span></div></details>
<p>There are <code>n</code> children with <code>ratings</code>. Each child gets at least one candy, and a child with a higher rating than an adjacent child must get more candies. Return the minimum total candies.</p>
<h2>Example 1</h2><pre>Input:  ratings = [1,0,2]
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  ratings = [1,2,2]
Output: 4</pre>
<h2>Constraints</h2><ul><li>n == ratings.length</li><li>1 &lt;= n &lt;= 2*10^4</li><li>0 &lt;= ratings[i] &lt;= 2*10^4</li></ul>
