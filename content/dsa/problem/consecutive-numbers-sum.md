<h1>Consecutive Numbers Sum</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span></div></details>
<p>Given an integer <code>n</code>, return the number of ways to write it as a sum of consecutive positive integers.</p>
<h2>Example 1</h2><pre>Input:  n = 5
Output: 2
Explanation: 5 = 5 = 2 + 3.</pre>
<h2>Example 2</h2><pre>Input:  n = 9
Output: 3
Explanation: 9 = 9 = 4 + 5 = 2 + 3 + 4.</pre>
<h2>Example 3</h2><pre>Input:  n = 15
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 10^9</li></ul>
