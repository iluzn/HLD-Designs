<h1>Path Sum III</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Prefix Sum</span></div></details>
<p>Given the <code>root</code> of a binary tree and an integer <code>targetSum</code> (passed as <code>k</code>), return the number of downward paths (parent to child, any start and end node) whose node values sum to <code>k</code>.</p>
<h2>Example 1</h2><pre>Input:  root = [10,5,-3,3,2,null,11,3,-2,null,1], k = 8
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  root = [5,4,8,11,null,13,4,7,2,null,null,5,1], k = 22
Output: 3</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 1000</li><li>-10^9 &lt;= Node.val &lt;= 10^9</li><li>-1000 &lt;= k &lt;= 1000</li></ul>
