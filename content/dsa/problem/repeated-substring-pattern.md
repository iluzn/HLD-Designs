<h1>Repeated Substring Pattern</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">String Matching</span></div></details>
<p>Given a string <code>s</code>, return <code>true</code> if it can be built by repeating a substring of it multiple times.</p>
<h2>Example 1</h2><pre>Input:  s = "abab"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "aba"
Output: false</pre>
<h2>Example 3</h2><pre>Input:  s = "abcabcabc"
Output: true</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^4</li><li>s consists of lowercase English letters.</li></ul>
