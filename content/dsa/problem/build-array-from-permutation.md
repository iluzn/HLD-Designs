<h1>Build Array from Permutation</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Simulation</span></div></details>
<p>Given a zero-based permutation <code>nums</code>, build an array <code>ans</code> of the same length where <code>ans[i] = nums[nums[i]]</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [0,2,1,5,3,4]
Output: [0,1,2,4,5,3]</pre>
<h2>Example 2</h2><pre>Input:  nums = [5,0,1,2,3,4]
Output: [4,5,0,1,2,3]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 1000</li><li>nums is a permutation of 0..n-1</li></ul>
