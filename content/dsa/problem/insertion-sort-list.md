<h1>Insertion Sort List</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Sorting</span></div></details>
<p>Given the <code>head</code> of a singly linked list, sort it in <strong>ascending</strong> order using insertion sort and return the sorted list.</p>
<h2>Example 1</h2><pre>Input:  head = [4,2,1,3]
Output: [1,2,3,4]</pre>
<h2>Example 2</h2><pre>Input:  head = [-1,5,3,4,0]
Output: [-1,0,3,4,5]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 5000</li><li>-5000 &lt;= Node.val &lt;= 5000</li></ul>
