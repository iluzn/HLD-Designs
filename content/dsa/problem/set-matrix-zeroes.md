<h1>Set Matrix Zeroes</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>m x n</code> matrix, if an element is 0 set its entire row and column to 0. LeetCode mutates in place; here return the modified matrix.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]</pre>
<h2>Example 2</h2><pre>Input:  grid = [[0,1],[1,1]]
Output: [[0,0],[0,1]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 200</li><li>-2^31 &lt;= matrix[i][j] &lt;= 2^31 - 1</li></ul>
