<h1>Find the Duplicate Number</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given an array <code>nums</code> of <code>n+1</code> integers where each value is in <code>[1, n]</code>, exactly one value is repeated. Return the repeated number without modifying the array and using O(1) extra space.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,3,4,2,2]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [3,1,3,4,2]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 10^5</li><li>nums.length == n + 1</li><li>Only one repeated number, but it may appear more than once.</li></ul>
