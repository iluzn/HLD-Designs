<h1>Shortest Subarray with Sum at Least K</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Prefix Sum</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Deque</span></div></details>
<p>Given an integer array <code>nums</code> and an integer <code>target</code>, return the length of the <strong>shortest</strong> non-empty contiguous subarray whose sum is at least <code>target</code>. If there is no such subarray, return <code>-1</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [1], target = 1
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2], target = 4
Output: -1</pre>
<h2>Example 3</h2><pre>Input:  nums = [2,-1,2], target = 3
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>-10^5 &lt;= nums[i] &lt;= 10^5</li><li>1 &lt;= target &lt;= 10^9</li></ul>
