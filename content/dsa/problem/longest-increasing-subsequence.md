<h1>Longest Increasing Subsequence</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an integer array <code>nums</code>, return the length of the longest strictly increasing subsequence.</p>
<h2>Example 1</h2><pre>Input:  nums = [10,9,2,5,3,7,101,18]
Output: 4
Explanation: [2,3,7,101] or [2,3,7,18].</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,1,0,3,2,3]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 2500</li><li>-10^4 &lt;= nums[i] &lt;= 10^4</li></ul>
