<h1>Top K Frequent Elements</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Heap</span><span class="lc-tag">Bucket Sort</span></div></details>
<p>Given an integer array <code>nums</code> and an integer <code>k</code>, return the <code>k</code> most frequent elements. <em>On this judge the answer is returned sorted in ascending order</em>, and the inputs guarantee the k most frequent elements are unambiguous.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,1,1,2,2,3], k = 2
Output: [1,2]</pre>
<h2>Example 2</h2><pre>Input:  nums = [1], k = 1
Output: [1]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>k is in the range [1, number of distinct elements]</li><li>The k most frequent elements are unique.</li></ul>
