<h1>Counting Bits</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Given an integer <code>n</code>, return an array <code>ans</code> of length <code>n + 1</code> where <code>ans[i]</code> is the number of <code>1</code>-bits in the binary representation of <code>i</code>.</p>
<h2>Example 1</h2><pre>Input:  n = 2
Output: [0,1,1]</pre>
<h2>Example 2</h2><pre>Input:  n = 5
Output: [0,1,1,2,1,2]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= n &lt;= 10^5</li></ul>
