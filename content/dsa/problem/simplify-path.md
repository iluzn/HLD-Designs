<h1>Simplify Path</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Stack</span></div></details>
<p>Given an absolute Unix-style path <code>path</code>, return its simplified canonical path. Handle <code>.</code> (current), <code>..</code> (parent), and redundant slashes.</p>
<h2>Example 1</h2><pre>Input:  path = "/home/"
Output: "/home"</pre>
<h2>Example 2</h2><pre>Input:  path = "/../"
Output: "/"</pre>
<h2>Example 3</h2><pre>Input:  path = "/a/./b/../../c/"
Output: "/c"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= path.length &lt;= 3000</li><li>path begins with a single slash.</li></ul>
