<h1>Subarrays with K Different Integers</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Counting</span></div></details>
<p>Given an integer array <code>nums</code> and an integer <code>target</code> (= k), return the number of contiguous subarrays that contain <strong>exactly</strong> <code>target</code> distinct integers.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,1,2,3], target = 2
Output: 7</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,1,3,4], target = 3
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 2*10^4</li><li>1 &lt;= nums[i] &lt;= nums.length</li><li>1 &lt;= target &lt;= nums.length</li></ul>
