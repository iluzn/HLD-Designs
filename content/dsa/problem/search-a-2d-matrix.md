<h1>Search a 2D Matrix</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Matrix</span></div></details>
<p>You are given an <code>m x n</code> matrix where each row is sorted ascending and the first integer of each row is greater than the last integer of the previous row. Return <code>true</code> if <code>target</code> is present.</p>
<h2>Example 1</h2><pre>Input:  matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
Output: true</pre>
<h2>Example 2</h2><pre>Input:  matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 100</li><li>-10^4 &lt;= matrix[i][j], target &lt;= 10^4</li></ul>
