<h1>Multiply Strings</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Math</span><span class="lc-tag">Simulation</span></div></details>
<p>Given two non-negative integers <code>num1</code> and <code>num2</code> represented as strings, return the product, also as a string. Do not use any built-in big-integer library or convert the inputs directly to an integer.</p>
<h2>Example 1</h2><pre>Input:  num1 = "2", num2 = "3"
Output: "6"</pre>
<h2>Example 2</h2><pre>Input:  num1 = "123", num2 = "456"
Output: "56088"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= num1.length, num2.length &lt;= 200</li><li>The numbers contain only digits and have no leading zero (except "0").</li></ul>
