<h1>Valid Parentheses</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Stack</span></div></details>
<p>Given a string <code>s</code> containing just <code>()[]{}</code>, determine if the input is valid: brackets are closed by the same type in the correct order.</p>
<h2>Example 1</h2><pre>Input:  s = "()[]{}"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "([)]"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^4</li><li>s consists of brackets only.</li></ul>
