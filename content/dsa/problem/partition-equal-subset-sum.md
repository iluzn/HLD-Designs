<h1>Partition Equal Subset Sum</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an array <code>nums</code> of positive integers, return <code>true</code> if it can be split into two subsets with equal sum.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,5,11,5]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3,5]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 200</li><li>1 &lt;= nums[i] &lt;= 100</li></ul>
