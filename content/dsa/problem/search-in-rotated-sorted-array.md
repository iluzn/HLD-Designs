<h1>Search in Rotated Sorted Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given a rotated ascending array of distinct integers <code>nums</code> and a <code>target</code>, return its index or <code>-1</code>. Must run in O(log n).</p>
<h2>Example 1</h2><pre>Input:  nums = [4,5,6,7,0,1,2], target = 0
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  nums = [4,5,6,7,0,1,2], target = 3
Output: -1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 5000</li><li>All values distinct.</li><li>nums is a rotation of an ascending array.</li></ul>
