<h1>Maximal Network Rank</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">Array</span></div></details>
<p>There are <code>n</code> cities and some undirected roads given as <code>edges</code>. The network rank of two different cities is the total number of roads directly connected to either city; a road connecting both counts only once. Return the maximum network rank over all pairs of cities.</p>
<h2>Example 1</h2><pre>Input:  n = 4, edges = [[0,1],[0,3],[1,2],[1,3]]
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  n = 5, edges = [[0,1],[0,3],[1,2],[1,3],[2,3],[2,4]]
Output: 5</pre>
<h2>Constraints</h2><ul><li>2 &lt;= n &lt;= 100</li><li>No repeated roads and no self-loops.</li></ul>
