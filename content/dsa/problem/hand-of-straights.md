<h1>Hand of Straights</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>hand</code> of card values and a group size <code>k</code> (LeetCode: groupSize), return <code>true</code> if the cards can be rearranged into groups of <code>k</code> consecutive cards.</p>
<h2>Example 1</h2><pre>Input:  hand = [1,2,3,6,2,3,4,7,8], k = 3
Output: true</pre>
<h2>Example 2</h2><pre>Input:  hand = [1,2,3,4,5], k = 4
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= hand.length &lt;= 10^4</li><li>0 &lt;= hand[i] &lt;= 10^9</li><li>1 &lt;= k</li></ul>
