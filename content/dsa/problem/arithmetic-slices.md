<h1>Arithmetic Slices</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>An arithmetic slice is a contiguous subarray of length at least 3 with a constant difference between consecutive elements. Return the number of arithmetic slices in <code>nums</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,4]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,3,5,7,9]
Output: 6</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,2,3,8,9,10]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 5000</li><li>-1000 &lt;= nums[i] &lt;= 1000</li></ul>
