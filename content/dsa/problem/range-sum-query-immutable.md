<h1>Range Sum Query - Immutable</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Prefix Sum</span><span class="lc-tag">Design</span></div></details>
<p>Given an integer array <code>nums</code>, handle multiple queries of the sum of the elements between indices <code>left</code> and <code>right</code> inclusive. Implement <code>NumArray(nums)</code> and <code>sumRange(left, right)</code> which returns the sum <code>nums[left] + ... + nums[right]</code>.</p>
<h2>Example 1</h2><pre>Input:  operations = ["NumArray","sumRange","sumRange","sumRange"], values = [[-2,0,3,-5,2,-1],[0,2],[2,5],[0,5]]
Output: [null,1,-1,-3]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^4</li><li>-10^5 &lt;= nums[i] &lt;= 10^5</li><li>0 &lt;= left &lt;= right &lt; nums.length</li><li>At most 10^4 calls to sumRange.</li></ul>
