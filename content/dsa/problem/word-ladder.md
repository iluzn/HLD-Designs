<h1>Word Ladder</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">BFS</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given <code>beginWord</code>, <code>endWord</code>, and a <code>wordList</code>, return the number of words in the shortest transformation sequence from begin to end (changing one letter at a time, each intermediate word in the list), or 0 if none. The count includes both endpoints.</p>
<h2>Example 1</h2><pre>Input:  beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log","cog"]
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log"]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= word length &lt;= 10</li><li>1 &lt;= wordList.length &lt;= 5000</li><li>All words have the same length.</li></ul>
