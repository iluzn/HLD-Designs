<h1>Decode Ways</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>A message of digits is encoded to letters using the mapping <code>'A' -&gt; 1</code>, ..., <code>'Z' -&gt; 26</code>. Given a digit string <code>s</code>, return the number of ways to decode it. The answer fits in a 32-bit integer.</p>
<h2>Example 1</h2><pre>Input:  s = "12"
Output: 2
Explanation: "AB" (1 2) or "L" (12).</pre>
<h2>Example 2</h2><pre>Input:  s = "226"
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  s = "06"
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 100</li><li>s contains only digits and may contain leading zeros.</li></ul>
