<h1>Happy Number</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Two Pointers</span></div></details>
<p>A number is <em>happy</em> if repeatedly replacing it by the sum of the squares of its digits eventually reaches 1. Numbers that never reach 1 loop forever. Return <code>true</code> if <code>n</code> is happy.</p>
<h2>Example 1</h2><pre>Input:  n = 19
Output: true
Explanation: 1+81=82, 64+4=68, 36+64=100, 1 -> happy.</pre>
<h2>Example 2</h2><pre>Input:  n = 2
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2^31 - 1</li></ul>
