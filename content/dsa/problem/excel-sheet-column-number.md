<h2>Example 1</h2><pre>Input:  s = "A"
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  s = "AB"
Output: 28</pre>
<h2>Example 3</h2><pre>Input:  s = "ZY"
Output: 701</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 7</li><li>s consists of uppercase English letters and is in the range ["A", "FXSHRXW"].</li></ul>
