<h1>Wiggle Subsequence</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Greedy</span></div></details>
<p>A wiggle sequence has strictly alternating positive and negative differences between successive numbers. Given <code>nums</code>, return the length of the longest wiggle subsequence.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,7,4,9,2,5]
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,17,5,10,13,15,10,5,16,8]
Output: 7</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,2,3,4,5]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 1000</li><li>0 &lt;= nums[i] &lt;= 1000</li></ul>
