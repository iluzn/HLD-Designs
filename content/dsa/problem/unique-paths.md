<h1>Unique Paths</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Combinatorics</span></div></details>
<p>A robot sits at the top-left of an <code>m x n</code> grid and can only move <strong>right</strong> or <strong>down</strong>. Return the number of distinct paths to the bottom-right corner.</p>
<h2>Example 1</h2><pre>Input:  m = 3, n = 7
Output: 28</pre>
<h2>Example 2</h2><pre>Input:  m = 3, n = 2
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 100</li></ul>
