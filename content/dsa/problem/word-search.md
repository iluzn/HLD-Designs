<h1>Word Search</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Backtracking</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>m x n</code> grid of characters <code>board</code> and a string <code>word</code>, return <code>true</code> if the word can be formed from sequentially adjacent (horizontally or vertically) cells, using each cell at most once.</p>
<h2>Example 1</h2><pre>Input:  board = ["ABCE","SFCS","ADEE"], word = "ABCCED"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  board = ["ABCE","SFCS","ADEE"], word = "ABCB"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 6</li><li>1 &lt;= word.length &lt;= 15</li></ul>
