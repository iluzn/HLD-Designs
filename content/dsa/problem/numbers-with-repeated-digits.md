<h1>Numbers With Repeated Digits</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an integer <code>n</code>, return the count of positive integers in the range <code>[1, n]</code> that have at least one repeated digit.</p>
<h2>Example 1</h2><pre>Input:  n = 20
Output: 1
Explanation: Only 11 has a repeated digit.</pre>
<h2>Example 2</h2><pre>Input:  n = 100
Output: 10</pre>
<h2>Example 3</h2><pre>Input:  n = 1000
Output: 262</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 10^9</li></ul>
