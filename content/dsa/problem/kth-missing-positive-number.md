<h1>Kth Missing Positive Number</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given a strictly increasing array of positive integers <code>nums</code> and an integer <code>target</code> (= k), return the <code>target</code>-th positive integer that is <strong>missing</strong> from the array.</p>
<h2>Example 1</h2><pre>Input:  nums = [2,3,4,7,11], target = 5
Output: 9</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3,4], target = 2
Output: 6</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 1000</li><li>1 &lt;= nums[i] &lt;= 1000</li><li>nums is strictly increasing.</li><li>1 &lt;= target &lt;= 1000</li></ul>
