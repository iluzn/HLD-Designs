<h1>Course Schedule</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">Topological Sort</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span></div></details>
<p>There are <code>n</code> courses labeled 0..n-1. <code>edges[i] = [a, b]</code> means you must take course <code>b</code> before course <code>a</code>. Return <code>true</code> if you can finish all courses.</p>
<h2>Example 1</h2><pre>Input:  n = 2, edges = [[1,0]]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  n = 2, edges = [[1,0],[0,1]]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2000</li><li>0 &lt;= edges.length &lt;= 5000</li></ul>
