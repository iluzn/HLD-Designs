<h1>Cherry Pickup II</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Matrix</span></div></details>
<p>Given a <code>grid</code> where each cell holds a number of cherries, two robots start at the top-left <code>(0,0)</code> and top-right <code>(0, cols-1)</code>. Each robot moves down one row at a time to a diagonally-left, straight-down, or diagonally-right cell. When a robot passes a cell it collects its cherries; if both robots are on the same cell only one collection counts. Return the maximum cherries both robots can collect together after reaching the bottom row.</p>
<h2>Example 1</h2><pre>Input:  grid = [[3,1,1],[2,5,1],[1,5,5],[2,1,1]]
Output: 24</pre>
<h2>Example 2</h2><pre>Input:  grid = [[5]]
Output: 5</pre>
<h2>Constraints</h2><ul><li>2 &lt;= rows, cols &lt;= 70</li><li>0 &lt;= grid[i][j] &lt;= 100</li></ul>
