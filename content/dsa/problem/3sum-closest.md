<h1>3Sum Closest</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code> and a value <code>target</code>, find three integers whose sum is closest to <code>target</code> and return that sum.</p>
<h2>Example 1</h2><pre>Input:  nums = [-1,2,1,-4], target = 1
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,0,0], target = 1
Output: 0</pre>
<h2>Constraints</h2><ul><li>3 &lt;= nums.length &lt;= 500</li><li>-1000 &lt;= nums[i] &lt;= 1000</li><li>-10^4 &lt;= target &lt;= 10^4</li></ul>
