<h1>Number of Good Leaf Nodes Pairs</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span></div></details>
<p>Given the <code>root</code> of a binary tree and an integer <code>distance</code>, a pair of two different <strong>leaf</strong> nodes is <em>good</em> if the length of the shortest path between them (in edges) is less than or equal to <code>distance</code>. Return the number of good leaf node pairs.</p>
<h2>Example 1</h2><pre>Input:  root = [1,2,3,null,4], distance = 3
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  root = [1,2,3,4,5,6,7], distance = 3
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  root = [7,1,4,6,null,5,3,null,null,null,null,null,null,2], distance = 3
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 2^10</li><li>1 &lt;= Node.val &lt;= 100</li><li>1 &lt;= distance &lt;= 10.</li></ul>
