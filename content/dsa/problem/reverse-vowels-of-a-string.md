<h1>Reverse Vowels of a String</h1>

<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given a string <code>s</code>, reverse only the vowels (<code>a e i o u</code>, both cases) and return the result.</p>
<h2>Example 1</h2><pre>Input:  s = "hello"
Output: "holle"</pre>
<h2>Example 2</h2><pre>Input:  s = "leetcode"
Output: "leotcede"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 3*10^5</li></ul>
