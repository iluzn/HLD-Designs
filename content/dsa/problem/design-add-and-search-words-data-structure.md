<p>Design a data structure with <code>addWord(word)</code> and <code>search(word)</code>. In search, a <code>.</code> can match any single letter. search returns whether any added word matches.</p>
<h2>Example 1</h2><pre>Input:  operations = ["WordDictionary","addWord","addWord","addWord","search","search","search","search"], values = [[],["bad"],["dad"],["mad"],["pad"],["bad"],[".ad"],["b.."]]
Output: [null,null,null,null,false,true,true,true]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= word.length &lt;= 25</li><li>Added words are lowercase letters; search words may contain ".".</li><li>At most 10^4 calls.</li></ul>
