<h1>Find Target Indices After Sorting the Array</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code> and an integer <code>target</code>, first sort <code>nums</code> in non-decreasing order. Return the list of indices <code>i</code> (in increasing order) such that after sorting <code>nums[i] == target</code>. If <code>target</code> is not present, return an empty list.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,5,2,3], target = 2
Output: [1,2]</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,5,2,3], target = 3
Output: [3]</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,2,5,2,3], target = 5
Output: [4]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 100</li><li>1 &lt;= nums[i], target &lt;= 100</li></ul>
