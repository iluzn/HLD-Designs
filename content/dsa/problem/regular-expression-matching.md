<h1>Regular Expression Matching</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Recursion</span></div></details>
<p>Given a string <code>s</code> (first line) and a pattern <code>p</code> (second line) supporting <code>.</code> (any single char) and <code>*</code> (zero or more of the preceding element), return <code>true</code> if the pattern matches the entire string.</p>
<h2>Example 1</h2><pre>Input:  s = "aa", p = "a*"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "ab", p = ".*"
Output: true</pre>
<h2>Example 3</h2><pre>Input:  s = "mississippi", p = "mis*is*p*."
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 20</li><li>1 &lt;= p.length &lt;= 20</li><li>Each * has a valid preceding element.</li></ul>
