<h1>Minimum Time to Complete Trips</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given an array <code>nums</code> where <code>nums[i]</code> is the time a bus takes for one trip, and an integer <code>target</code> (the total number of trips required by all buses combined), return the minimum time needed for the buses to complete at least <code>target</code> trips. Buses run independently and in parallel.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3], target = 5
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [2], target = 1
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= 10^7</li><li>1 &lt;= target &lt;= 10^7</li></ul>
