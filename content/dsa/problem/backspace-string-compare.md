<h1>Backspace String Compare</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Stack</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given two strings <code>s</code> and <code>t</code>, return <code>true</code> if they are equal after typing them into empty text editors. <code>#</code> is a backspace character.</p>
<h2>Example 1</h2><pre>Input:  s = "ab#c", t = "ad#c"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "a##c", t = "#a#c"
Output: true</pre>
<h2>Example 3</h2><pre>Input:  s = "a#c", t = "b"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length, t.length &lt;= 200</li><li>s and t contain lowercase letters and #.</li></ul>
