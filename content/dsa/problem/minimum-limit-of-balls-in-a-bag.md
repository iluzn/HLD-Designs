<h1>Minimum Limit of Balls in a Bag</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given an array <code>nums</code> of bag sizes and an integer <code>target</code> (the maximum number of split operations allowed), each operation splits one bag into two smaller non-negative bags. Return the minimum possible penalty, where the penalty is the maximum number of balls in any bag after all operations.</p>
<h2>Example 1</h2><pre>Input:  nums = [9], target = 2
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [2,4,8,2], target = 4
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= 10^9</li><li>1 &lt;= target &lt;= 10^9</li></ul>
