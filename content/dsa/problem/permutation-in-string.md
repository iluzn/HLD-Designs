<h1>Permutation in String</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given two strings <code>s1</code> and <code>s2</code>, return <code>true</code> if <code>s2</code> contains a permutation of <code>s1</code> as a contiguous substring. (Here the first line is s1, the second is s2.)</p>
<h2>Example 1</h2><pre>Input:  s1 = "ab", s2 = "eidbaooo"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s1 = "ab", s2 = "eidboaoo"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s1.length, s2.length &lt;= 10^4</li><li>s1 and s2 consist of lowercase English letters.</li></ul>
