<h1>Redundant Connection</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">Union Find</span></div></details>
<p>A tree on nodes 1..n had one extra edge added, creating exactly one cycle. Given the <code>edges</code> in input order, return the edge that can be removed so the result is a tree. If several qualify, return the one that appears last in the input.</p>
<h2>Example 1</h2><pre>Input:  edges = [[1,2],[1,3],[2,3]]
Output: [2,3]</pre>
<h2>Example 2</h2><pre>Input:  edges = [[1,2],[2,3],[3,4],[1,4],[1,5]]
Output: [1,4]</pre>
<h2>Constraints</h2><ul><li>3 &lt;= n &lt;= 1000</li><li>The graph is a tree plus exactly one extra edge.</li></ul>
