<h1>Kth Largest Element in a Stream</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Heap</span><span class="lc-tag">Design</span><span class="lc-tag">Binary Search Tree</span></div></details>
<p>Design a class to find the <code>k</code>th largest element in a stream (the kth largest in sorted order, not the kth distinct). Implement <code>KthLargest(k, nums)</code> which initializes with an integer <code>k</code> and a stream of integers <code>nums</code>, and <code>add(val)</code> which appends <code>val</code> and returns the kth largest element so far.</p>
<h2>Example 1</h2><pre>Input:  operations = ["KthLargest","add","add","add","add","add"], values = [[3,[4,5,8,2]],[3],[5],[10],[9],[4]]
Output: [null,4,5,5,8,8]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= k &lt;= 10^4</li><li>At least k elements exist when add is queried.</li><li>At most 10^4 calls to add.</li></ul>
