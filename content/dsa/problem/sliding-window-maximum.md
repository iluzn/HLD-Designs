<h1>Sliding Window Maximum</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Heap</span><span class="lc-tag">Monotonic Queue</span></div></details>
<p>Given an array <code>nums</code> and a window size <code>k</code>, the window slides from left to right one position at a time. Return an array of the maximum value in each window.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,3,-1,-3,5,3,6,7], k = 3
Output: [3,3,5,5,6,7]</pre>
<h2>Example 2</h2><pre>Input:  nums = [1], k = 1
Output: [1]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= k &lt;= nums.length</li></ul>
