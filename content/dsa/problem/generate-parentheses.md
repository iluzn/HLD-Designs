<h1>Generate Parentheses</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Backtracking</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given <code>n</code> pairs of parentheses, return all combinations of well-formed parentheses, in any order.</p>
<h2>Example 1</h2><pre>Input:  n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]</pre>
<h2>Example 2</h2><pre>Input:  n = 1
Output: ["()"]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 8</li></ul>
