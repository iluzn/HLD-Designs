<h1>Median of Two Sorted Arrays</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Divide and Conquer</span></div></details>
<p>Given two sorted arrays <code>nums1</code> and <code>nums2</code> of size m and n respectively, return the <strong>median</strong> of the two sorted arrays. The overall run time complexity should be O(log(m+n)).</p><p>The median of an array of length L is the (L/2)-th element if L is odd, or the average of the (L/2-1)-th and (L/2)-th elements if L is even (0-indexed, integer division). For this problem, return the integer part (floor) of the median.</p>
<h2>Example 1</h2><pre>Input:  nums1 = [1,3], nums2 = [2]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums1 = [1,2], nums2 = [3,4]
Output: 2</pre>
<h2>Constraints</h2><ul><li>0 &lt;= m, n &lt;= 1000</li><li>1 &lt;= m + n</li><li>-10^6 &lt;= nums1[i], nums2[i] &lt;= 10^6</li><li>Both arrays are sorted in non-decreasing order.</li></ul>
