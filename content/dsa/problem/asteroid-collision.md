<h1>Asteroid Collision</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Stack</span><span class="lc-tag">Simulation</span></div></details>
<p>Given <code>nums</code> of asteroids (absolute value = size, sign = direction: positive right, negative left), return the state after all collisions. Two moving toward each other: the smaller explodes; equal sizes both explode. Same direction never collide.</p>
<h2>Example 1</h2><pre>Input:  asteroids = [5,10,-5]
Output: [5,10]</pre>
<h2>Example 2</h2><pre>Input:  asteroids = [8,-8]
Output: []</pre>
<h2>Example 3</h2><pre>Input:  asteroids = [10,2,-5]
Output: [10]</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 10^4</li><li>-1000 &lt;= nums[i] &lt;= 1000</li><li>nums[i] != 0</li></ul>
