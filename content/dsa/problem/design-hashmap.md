<h1>Design HashMap</h1>

<p>Design a HashMap without using any built-in hash table libraries. Implement <code>MyHashMap</code> with <code>put(key, value)</code>, <code>get(key)</code> (returns the value or <code>-1</code> if the key is absent), and <code>remove(key)</code>.</p>

Output: [null,null,null,1,-1,null,1,null,-1]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= key, value &lt;= 10^6</li><li>At most 10^4 calls to put, get, and remove.</li></ul>
