<h1>Kth Largest Element in an Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Heap</span><span class="lc-tag">Quickselect</span></div></details>
<p>Given an integer array <code>nums</code> and an integer <code>k</code> (passed as <code>target</code>), return the <code>k</code>-th largest element in the array (in sorted order, not distinct).</p>
<h2>Example 1</h2><pre>Input:  nums = [3,2,1,5,6,4], k = 2
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  nums = [3,2,3,1,2,4,5,5,6], k = 4
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= k &lt;= nums.length &lt;= 10^4</li><li>-10^4 &lt;= nums[i] &lt;= 10^4</li></ul>
