<h1>Max Consecutive Ones</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span></div></details>
<p>Given a binary array <code>nums</code>, return the maximum number of consecutive <code>1</code>s.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,1,0,1,1,1]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,0,1,1,0,1]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>nums[i] is 0 or 1</li></ul>
