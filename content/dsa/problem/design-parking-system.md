<h1>Design Parking System</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Design</span><span class="lc-tag">Counting</span><span class="lc-tag">Simulation</span></div></details>
<p>Design a parking system for a lot with three kinds of spaces: big, medium, and small, with a fixed number of each. Implement <code>ParkingSystem(big, medium, small)</code> and <code>addCar(carType)</code> where <code>carType</code> is 1 (big), 2 (medium), or 3 (small). Return <code>true</code> if there is an available slot of that type (and park the car), otherwise <code>false</code>.</p>
<h2>Example 1</h2><pre>Input:  operations = ["ParkingSystem","addCar","addCar","addCar","addCar"], values = [[1,1,0],[1],[2],[3],[1]]
Output: [null,true,true,false,false]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= big, medium, small &lt;= 1000</li><li>carType is 1, 2, or 3.</li><li>At most 1000 calls to addCar.</li></ul>
