<h1>Longest Univalue Path</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, return the length of the longest path where every node on the path has the <strong>same value</strong>. The length is measured in <strong>edges</strong>. The path need not pass through the root.</p>
<h2>Example 1</h2><pre>Input:  root = [5,4,5,1,1,null,5]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  root = [1,4,5,4,4,null,5]
Output: 2</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^4</li><li>-1000 &lt;= Node.val &lt;= 1000</li></ul>
