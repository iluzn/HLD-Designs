<h1>Maximum Sum BST in Binary Tree</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Binary Search Tree</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given the <code>root</code> of a binary tree, return the maximum sum of all keys of any sub-tree which is also a <strong>Binary Search Tree</strong> (BST). A valid BST requires every node in the left subtree to be strictly less than the node, and every node in the right subtree to be strictly greater. If no sub-tree is a BST with positive sum, return <code>0</code>.</p>
<h2>Example 1</h2><pre>Input:  root = [1,4,3,2,4,2,5,null,null,null,null,null,null,4,6]
Output: 20</pre>
<h2>Example 2</h2><pre>Input:  root = [4,3,null,1,2]
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  root = [-4,-2,-5]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 4*10^4</li><li>-4*10^4 &lt;= Node.val &lt;= 4*10^4.</li></ul>
