<h1>Walls and Gates</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">BFS</span><span class="lc-tag">Matrix</span></div></details>
<p>Fill each empty room with the distance to its nearest gate. <code>-1</code> is a wall, <code>0</code> is a gate, and <code>2147483647</code> (INF) is an empty room. If a room cannot reach a gate, leave it as INF. Return the filled grid.</p>
<h2>Example 1</h2><pre>Input:  rooms = [[2147483647,-1,0,2147483647],[2147483647,2147483647,2147483647,-1],[2147483647,-1,2147483647,-1],[0,-1,2147483647,2147483647]]
Output: [[3,-1,0,1],[2,2,1,-1],[1,-1,2,-1],[0,-1,3,4]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 250</li><li>Each cell is -1, 0, or 2147483647.</li></ul>
