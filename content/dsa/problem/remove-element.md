<h2>Example 1</h2><pre>Input:  nums = [3,2,2,3], target = 3
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,1,2,2,3,0,4,2], target = 2
Output: 5</pre>
<h2>Constraints</h2><ul><li>0 &lt;= nums.length &lt;= 100</li><li>0 &lt;= nums[i], target &lt;= 50</li></ul>
