<h1>Third Maximum Number</h1>

<p>Given an integer array <code>nums</code>, return the third distinct maximum value. If it does not exist, return the maximum value.</p>
<h2>Example 1</h2><pre>Input:  nums = [3,2,1]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2]
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  nums = [2,2,3,1]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^4</li><li>-2^31 &lt;= nums[i] &lt;= 2^31 - 1</li></ul>
