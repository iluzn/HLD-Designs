<h1>Trapping Rain Water</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Stack</span></div></details>
<p>Given <code>n</code> non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.</p>
<h2>Example 1</h2><pre>Input:  height = [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  height = [4,2,0,3,2,5]
Output: 9</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2*10^4</li><li>0 &lt;= height[i] &lt;= 10^5</li></ul>
