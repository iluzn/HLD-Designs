<h1>Count Good Nodes in Binary Tree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span></div></details>
<p>A node is <em>good</em> if no node on the path from the root to it has a greater value. Given the <code>root</code>, return the number of good nodes.</p>
<h2>Example 1</h2><pre>Input:  root = [3,1,4,3,null,1,5]
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  root = [3,3,null,4,2]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^5</li><li>-10^4 &lt;= Node.val &lt;= 10^4</li></ul>
