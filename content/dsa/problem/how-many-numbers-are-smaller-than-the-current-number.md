<p>For each <code>nums[i]</code>, count how many <code>nums[j]</code> (j != i) are strictly smaller. Return the counts as an array.</p>
<h2>Example 1</h2><pre>Input:  nums = [8,1,2,2,3]
Output: [4,0,1,1,3]</pre>
<h2>Example 2</h2><pre>Input:  nums = [6,5,4,8]
Output: [2,1,0,3]</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 500</li><li>0 &lt;= nums[i] &lt;= 100</li></ul>
