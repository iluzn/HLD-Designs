<p>Design a circular queue using a fixed-size array. Implement <code>MyCircularQueue(k)</code> and: <code>enQueue(value)</code> (returns true on success), <code>deQueue()</code> (returns true on success), <code>Front()</code> and <code>Rear()</code> (return the front/rear value or <code>-1</code> if empty), <code>isEmpty()</code>, and <code>isFull()</code>.</p>

Output: [null,true,true,true,false,3,true,true,true,4]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= k &lt;= 1000</li><li>0 &lt;= value &lt;= 1000</li><li>At most 3000 calls.</li></ul>
