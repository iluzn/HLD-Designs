<h1>Group Anagrams</h1>

<p>Given an array of strings <code>strs</code>, group the anagrams together. Return the groups in any order.</p>
<h2>Example 1</h2><pre>Input:  strs = ["eat","tea","tan","ate","nat","bat"]
Output: [["bat"],["nat","tan"],["ate","eat","tea"]]</pre>
<h2>Example 2</h2><pre>Input:  strs = [""]
Output: [[""]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= strs.length &lt;= 10^4</li><li>0 &lt;= strs[i].length &lt;= 100</li><li>strs[i] consists of lowercase English letters.</li></ul>
