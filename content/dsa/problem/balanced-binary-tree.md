<h1>Balanced Binary Tree</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span></div></details>
<p>Given a binary tree, determine if it is height-balanced — for every node, the heights of its two subtrees differ by at most 1.</p>
<h2>Example 1</h2><pre>Input:  root = [3,9,20,null,null,15,7]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  root = [1,2,2,3,3,null,null,4,4]
Output: false</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 5000</li><li>-10^4 &lt;= Node.val &lt;= 10^4</li></ul>
