<h1>Maximum Gap</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sorting</span><span class="lc-tag">Bucket Sort</span></div></details>
<p>Given an integer array <code>nums</code>, return the maximum difference between two successive elements once the array is sorted in ascending order. If the array contains fewer than two elements, return <code>0</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [3,6,9,1]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [10]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>0 &lt;= nums[i] &lt;= 10^9</li></ul>
