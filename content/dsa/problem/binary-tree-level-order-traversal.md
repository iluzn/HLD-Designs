<h1>Binary Tree Level Order Traversal</h1>

<p>Given the <code>root</code> of a binary tree, return its level-order traversal — a list of levels, each a list of node values from left to right. On this judge levels are shown separated by <code>|</code>.</p>
<h2>Example 1</h2><pre>Input:  root = [3,9,20,null,null,15,7]
Output: [[3],[9,20],[15,7]]</pre>
<h2>Example 2</h2><pre>Input:  root = []
Output: []</pre>
