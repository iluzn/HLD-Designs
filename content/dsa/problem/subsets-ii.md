<p>Given an integer array <code>nums</code> that may contain duplicates, return all possible <strong>unique</strong> subsets. The result may be in any order.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,2]
Output: [[],[1],[2],[1,2],[2,2],[1,2,2]]</pre>
<h2>Example 2</h2><pre>Input:  nums = [0]
Output: [[],[0]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10</li><li>-10 &lt;= nums[i] &lt;= 10</li></ul>
