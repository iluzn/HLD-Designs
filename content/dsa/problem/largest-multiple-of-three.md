<h1>Largest Multiple of Three</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Greedy</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code> of digits (each 0-9), form the largest possible multiple of three by concatenating some of the digits, using each at most once. Return that number as an integer. If no non-empty selection is a multiple of three, return <code>-1</code>. If the largest value you can form is <code>0</code> (only zeros usable), return <code>0</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [8,1,9]
Output: 981</pre>
<h2>Example 2</h2><pre>Input:  nums = [8,6,7,1,0]
Output: 8760</pre>
<h2>Example 3</h2><pre>Input:  nums = [1]
Output: -1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 7</li><li>0 &lt;= nums[i] &lt;= 9</li></ul>
