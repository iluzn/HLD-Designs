<h1>Valid Sudoku</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Matrix</span></div></details>
<p>Determine if a 9 x 9 Sudoku <code>board</code> is valid. Only the filled cells need to be checked: each row, each column, and each 3 x 3 sub-box must contain no repeated digit 1-9. Empty cells are <code>'.'</code>. The board need not be solvable.</p>
<h2>Example 1</h2><pre>Input:  board rows like "53..7...."
Output: true</pre>
<h2>Example 2</h2><pre>Input:  a board with two 8s in a column
Output: false</pre>
<h2>Constraints</h2><ul><li>board.length == 9</li><li>board[i].length == 9</li><li>board[i][j] is a digit 1-9 or '.'</li></ul>
