<h1>Reverse Pairs</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Merge Sort</span><span class="lc-tag">Binary Indexed Tree</span></div></details>
<p>Given an integer array <code>nums</code>, return the number of <strong>reverse pairs</strong>. A reverse pair is a pair <code>(i, j)</code> where <code>0 &lt;= i &lt; j &lt; nums.length</code> and <code>nums[i] &gt; 2 * nums[j]</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,3,2,3,1]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [2,4,3,5,1]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 5*10^4</li><li>-2^31 &lt;= nums[i] &lt;= 2^31 - 1</li></ul>
