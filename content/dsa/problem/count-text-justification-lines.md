<h1>Text Justification Line Count</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">String</span><span class="lc-tag">Greedy</span><span class="lc-tag">Simulation</span></div></details>
<p>Given an array of words <code>nums</code> (encoded as lengths) and a max width <code>target</code>, pack words into lines greedily (as many as fit with at least one space between them). Return the number of lines needed.</p>
<h2>Example 1</h2><pre>Input:  wordLengths = [4,2,3,7,1,1,1,3,2,2,1], maxWidth = 8
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  wordLengths = [1,1,1,1], maxWidth = 3
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 300</li><li>1 &lt;= nums[i] &lt;= target</li><li>1 &lt;= target &lt;= 100</li></ul>
