<h1>Longest Cycle in a Graph</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">DFS</span></div></details>
<p>You are given a directed graph of <code>n</code> nodes numbered from <code>0</code> to <code>n - 1</code>, where each node has <strong>at most one</strong> outgoing edge.</p><p>The graph is represented by a 0-indexed array <code>nums</code> of size <code>n</code>, where <code>nums[i]</code> indicates that there is a directed edge from node <code>i</code> to node <code>nums[i]</code>. If there is no outgoing edge from node <code>i</code>, then <code>nums[i] == -1</code>.</p><p>Return the length of the <strong>longest</strong> cycle in the graph. If no cycle exists, return <code>-1</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [3,3,4,2,3]
Output: 3
Explanation: Nodes 2 → 4 → 3 → 2 form a cycle of length 3.</pre>
<h2>Example 2</h2><pre>Input:  nums = [2,-1,3,1]
Output: -1
Explanation: No cycle exists.</pre>
<h2>Example 3</h2><pre>Input:  nums = [-1,4,-1,2,0,4]
Output: -1</pre>
<h2>Constraints</h2><ul><li>n == nums.length</li><li>1 &lt;= n &lt;= 10^5</li><li>-1 &lt;= nums[i] &lt; n</li><li>nums[i] != i (no self-loops)</li></ul>
