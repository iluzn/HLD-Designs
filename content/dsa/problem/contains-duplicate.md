<p>Given an integer array <code>nums</code>, return <code>true</code> if any value appears at least twice, and <code>false</code> if every element is distinct.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,1]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3,4]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>-10^9 &lt;= nums[i] &lt;= 10^9</li></ul>
