<h1>Minimum Number of K Consecutive Bit Flips</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Bit Manipulation</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Prefix Sum</span></div></details>
<p>You are given a binary array <code>nums</code> and an integer <code>target</code> (the flip window length <code>k</code>). In one operation you choose a contiguous subarray of length exactly <code>k</code> and flip every bit in it. Return the minimum number of operations needed to make every element <code>1</code>, or <code>-1</code> if it is impossible.</p>
<h2>Example 1</h2><pre>Input:  nums = [0,1,0], target = 1
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,1,0], target = 2
Output: -1</pre>
<h2>Example 3</h2><pre>Input:  nums = [0,0,0,1,0,1,1,0], target = 3
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= target &lt;= nums.length</li><li>nums[i] is 0 or 1.</li></ul>
