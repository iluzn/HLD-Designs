<h1>Number of Islands</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span><span class="lc-tag">Union Find</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>m x n</code> grid of <code>'1'</code> (land) and <code>'0'</code> (water), return the number of islands. An island is surrounded by water and formed by connecting adjacent land cells horizontally or vertically.</p>
<h2>Example 1</h2><pre>Input:  grid = ["11110","11010","11000","00000"]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  grid = ["11000","11000","00100","00011"]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 300</li><li>grid[i][j] is '0' or '1'.</li></ul>
