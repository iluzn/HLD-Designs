<h1>Max Area of Island</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>m x n</code> binary grid, return the area (number of cells) of the largest island. Cells connect horizontally or vertically. If there is no island, return 0.</p>
<h2>Example 1</h2><pre>Input:  grid = [[0,0,1,0],[1,1,1,0],[0,1,0,0]]
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  grid = [[0,0,0],[0,0,0]]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 50</li><li>grid[i][j] is 0 or 1.</li></ul>
