<h1>Scramble String</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>We can scramble a string <code>s</code> by recursively splitting it into two non-empty substrings and optionally swapping them, then scrambling each part. Given two strings <code>s</code> (as <code>s</code>) and <code>t</code>, return <code>true</code> if <code>t</code> is a scrambled string of <code>s</code>.</p>
<h2>Example 1</h2><pre>Input:  s = "great", t = "rgeat"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "abcde", t = "caebd"
Output: false</pre>
<h2>Example 3</h2><pre>Input:  s = "a", t = "a"
Output: true</pre>
<h2>Constraints</h2><ul><li>s.length == t.length</li><li>1 &lt;= s.length &lt;= 8</li><li>s and t consist of lowercase English letters.</li></ul>
