<h1>Find Median from Data Stream</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Heap</span><span class="lc-tag">Design</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Sorting</span></div></details>
<p>The median is the middle value in an ordered list; for an even-sized list it is the average of the two middle values. Design a structure that supports a stream of integers. Implement <code>MedianFinder</code> with <code>addNum(num)</code> to add an integer, and <code>findMedian()</code> to return the median of all elements so far. Medians are reported to <strong>one decimal place</strong> (e.g. <code>2.0</code>, <code>1.5</code>).</p>
<h2>Example 1</h2><pre>Input:  operations = ["MedianFinder","addNum","addNum","findMedian","addNum","findMedian"], values = [[],[1],[2],[],[3],[]]
Output: [null,null,null,1.5,null,2.0]</pre>
<h2>Constraints</h2><ul><li>-10^5 &lt;= num &lt;= 10^5</li><li>At most 5*10^4 calls to addNum and findMedian.</li><li>findMedian is only called on a non-empty structure.</li></ul>
