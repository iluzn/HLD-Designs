<h1>Delete the Middle Node of a Linked List</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given the <code>head</code> of a linked list, delete the <strong>middle</strong> node (the node at 0-indexed position <code>floor(n/2)</code>) and return the head. If the list has one node, return an empty list.</p>
<h2>Example 1</h2><pre>Input:  head = [1,3,4,7,1,2,6]
Output: [1,3,4,1,2,6]</pre>
<h2>Example 2</h2><pre>Input:  head = [1,2,3,4]
Output: [1,2,4]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^5</li><li>1 &lt;= Node.val &lt;= 10^5</li></ul>
