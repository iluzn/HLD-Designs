<h1>Merge Triplets to Form Target Triplet</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span></div></details>
<p>You are given <code>triplets</code> and a <code>target</code> triplet. A merge replaces the running triplet with the element-wise maximum of it and a chosen triplet. Return <code>true</code> if some sequence of merges can produce exactly <code>target</code>.</p>
<h2>Example 1</h2><pre>Input:  triplets = [[2,5,3],[1,8,4],[1,7,5]], target = [2,7,5]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  triplets = [[3,4,5],[4,5,6]], target = [3,2,5]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= triplets.length &lt;= 10^5</li><li>triplets[i].length == target.length == 3</li></ul>
