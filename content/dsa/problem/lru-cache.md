<h1>LRU Cache</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">Linked List</span><span class="lc-tag">Design</span></div></details>
<p>Design an LRU (Least Recently Used) cache with <code>get(key)</code> and <code>put(key, value)</code>, both O(1). The constructor takes a positive <code>capacity</code>. <code>get</code> returns the value or -1; <code>put</code> inserts/updates and evicts the least recently used key when over capacity.</p>
<h2>Example 1</h2><pre>Input:  operations = ["LRUCache","put","put","get","put","get","put","get","get","get"], values = [[2],[1,1],[2,2],[1],[3,3],[2],[4,4],[1],[3],[4]]
Output: [null,null,null,1,null,-1,null,-1,3,4]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= capacity &lt;= 3000</li><li>0 &lt;= key, value &lt;= 10^4</li><li>At most 2*10^5 calls.</li></ul>
