<p>You are given a large integer as an array of digits <code>digits</code> (most significant first). Increment it by one and return the resulting digits.</p>
<h2>Example 1</h2><pre>Input:  digits = [1,2,3]
Output: [1,2,4]</pre>
<h2>Example 2</h2><pre>Input:  digits = [9,9]
Output: [1,0,0]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= digits.length &lt;= 100</li><li>0 &lt;= digits[i] &lt;= 9</li><li>No leading zeros (except the number 0).</li></ul>
