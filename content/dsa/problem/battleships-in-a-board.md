<h1>Battleships in a Board</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">DFS</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>m x n</code> board where each cell is either <code>'X'</code> (part of a battleship) or <code>'.'</code> (empty), return the number of battleships. Battleships are placed horizontally or vertically, and no two battleships are adjacent (there is at least one empty cell between them).</p>
<h2>Example 1</h2><pre>Input:  board = ["X..X","...X","...X"]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  board = ["."]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 200</li><li>board[i][j] is 'X' or '.'.</li></ul>
