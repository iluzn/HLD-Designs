<h2>Example 1</h2><pre>Input:  n = 14
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  n = 8
Output: 4</pre>
<h2>Example 3</h2><pre>Input:  n = 0
Output: 0</pre>
<h2>Constraints</h2><ul><li>0 &lt;= n &lt;= 10^6</li></ul>
