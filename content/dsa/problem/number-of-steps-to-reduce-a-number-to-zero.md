<h1>Number of Steps to Reduce a Number to Zero</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Given an integer <code>n</code>, return the number of steps to reduce it to zero. In one step, if the number is even divide it by two, otherwise subtract one.</p>
<h2>Example 1</h2><pre>Input:  n = 14
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  n = 8
Output: 4</pre>
<h2>Example 3</h2><pre>Input:  n = 0
Output: 0</pre>
<h2>Constraints</h2><ul><li>0 &lt;= n &lt;= 10^6</li></ul>
