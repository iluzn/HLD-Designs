<h1>Next Greater Element II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Stack</span><span class="lc-tag">Monotonic Stack</span></div></details>
<p>Given a <strong>circular</strong> array <code>nums</code>, return an array where each position holds the next greater element (searching circularly), or <code>-1</code> if none exists.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,1]
Output: [2,-1,2]</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3,4,3]
Output: [2,3,4,-1,4]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^4</li><li>-10^9 &lt;= nums[i] &lt;= 10^9</li></ul>
