<h1>Longest Palindrome</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">String</span><span class="lc-tag">Greedy</span></div></details>
<p>Given a string <code>s</code> of lowercase and/or uppercase letters, return the length of the <strong>longest palindrome</strong> that can be built with those letters (letters are case-sensitive).</p>
<h2>Example 1</h2><pre>Input:  s = "abccccdd"
Output: 7</pre>
<h2>Example 2</h2><pre>Input:  s = "a"
Output: 1</pre>
<h2>Example 3</h2><pre>Input:  s = "bb"
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 2000</li><li>s consists of lowercase and/or uppercase English letters.</li></ul>
