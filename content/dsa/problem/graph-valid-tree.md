<h1>Graph Valid Tree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">Union Find</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span></div></details>
<p>Given <code>n</code> nodes labeled 0..n-1 and a list of undirected <code>edges</code>, return <code>true</code> if the graph is a valid tree (fully connected and acyclic).</p>
<h2>Example 1</h2><pre>Input:  n = 5, edges = [[0,1],[0,2],[0,3],[1,4]]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  n = 5, edges = [[0,1],[1,2],[2,3],[1,3],[1,4]]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 2000</li><li>0 &lt;= edges.length &lt;= 5000</li></ul>
