<h1>Jump Game II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an array <code>nums</code> where <code>nums[i]</code> is the maximum jump length from index <code>i</code>, return the minimum number of jumps to reach the last index. You can assume the last index is always reachable.</p>
<h2>Example 1</h2><pre>Input:  nums = [2,3,1,1,4]
Output: 2
Explanation: Jump 1 step to index 1, then 3 steps to the end.</pre>
<h2>Example 2</h2><pre>Input:  nums = [2,3,0,1,4]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^4</li><li>0 &lt;= nums[i] &lt;= 1000</li><li>The last index is reachable.</li></ul>
