<h1>Network Delay Time</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">Dijkstra</span><span class="lc-tag">Heap</span><span class="lc-tag">Shortest Path</span></div></details>
<p>You are given directed weighted edges <code>times</code> where <code>times[i] = [u, v, w]</code>, <code>n</code> nodes labeled 1..n, and a source <code>k</code>. Return the time for all nodes to receive a signal from <code>k</code>, or <code>-1</code> if some node is unreachable.</p>
<h2>Example 1</h2><pre>Input:  times = [[2,1,1],[2,3,1],[3,4,1]], n = 4, k = 2
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  times = [[1,2,1]], n = 2, k = 2
Output: -1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 100</li><li>1 &lt;= times.length &lt;= 6000</li><li>The graph is directed with positive weights.</li></ul>
