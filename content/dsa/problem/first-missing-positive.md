<h1>First Missing Positive</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given an unsorted array <code>nums</code>, return the smallest missing positive integer. Aim for O(n) time and O(1) extra space (use the array itself as a hash).</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,0]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [3,4,-1,1]
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  nums = [7,8,9,11,12]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>-2^31 &lt;= nums[i] &lt;= 2^31 - 1</li></ul>
