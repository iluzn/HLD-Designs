<h1>Find First and Last Position of Element in Sorted Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given a <strong>sorted</strong> array <code>nums</code> and a <code>target</code>, return <code>[first, last]</code> — the first and last index of <code>target</code>. Return <code>[-1, -1]</code> if it is not found. Must run in O(log n).</p>
<h2>Example 1</h2><pre>Input:  nums = [5,7,7,8,8,10], target = 8
Output: [3,4]</pre>
<h2>Example 2</h2><pre>Input:  nums = [5,7,7,8,8,10], target = 6
Output: [-1,-1]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= nums.length &lt;= 10^5</li><li>nums is sorted in non-decreasing order.</li></ul>
