<h1>Longest Increasing Path in a Matrix</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">DFS</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Matrix</span><span class="lc-tag">Memoization</span></div></details>
<p>Given an <code>m x n</code> integer matrix, return the length of the longest strictly increasing path. From a cell you may move up, down, left, or right (no diagonals or wrap-around).</p>
<h2>Example 1</h2><pre>Input:  matrix = [[9,9,4],[6,6,8],[2,1,1]]
Output: 4
Explanation: The path [1,2,6,9].</pre>
<h2>Example 2</h2><pre>Input:  matrix = [[3,4,5],[3,2,6],[2,2,1]]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= m, n &lt;= 200</li><li>0 &lt;= matrix[i][j] &lt;= 2^31 - 1</li></ul>
