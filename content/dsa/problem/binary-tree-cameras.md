<h1>Binary Tree Cameras</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Greedy</span></div></details>
<p>You are given the <code>root</code> of a binary tree. A camera placed on a node can monitor its parent, itself, and its immediate children. Return the <strong>minimum</strong> number of cameras needed to monitor every node of the tree.</p>
<h2>Example 1</h2><pre>Input:  root = [0,0,null,0,0]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  root = [0,0,null,0,null,0,null,null,0]
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 1000</li><li>Node.val == 0.</li></ul>
