<h1>Find Bottom Left Tree Value</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">BFS</span><span class="lc-tag">DFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, return the leftmost value in the <strong>last</strong> (deepest) row of the tree.</p>
<h2>Example 1</h2><pre>Input:  root = [2,1,3]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  root = [1,2,3,4,null,5,6,null,null,7]
Output: 7</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^4</li><li>-2^31 &lt;= Node.val &lt;= 2^31 - 1</li></ul>
