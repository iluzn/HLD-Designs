<h1>Minimum Number of Taps to Open to Water a Garden</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Greedy</span></div></details>
<p>A garden is a one-dimensional interval <code>[0, k]</code>. There are <code>nums.length == k + 1</code> taps, one at each integer point <code>i</code>. Tap <code>i</code> waters the area <code>[i - nums[i], i + nums[i]]</code>. Return the minimum number of taps to open so the whole garden is watered, or <code>-1</code> if it is impossible.</p>
<h2>Example 1</h2><pre>Input:  ranges = [3,4,1,1,0,0], n = 5
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  ranges = [0,0,0,0], n = 3
Output: -1</pre>
<h2>Example 3</h2><pre>Input:  ranges = [1,2,1,0,2,1,0,1], n = 7
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= k &lt;= 10^4</li><li>nums.length == k + 1</li><li>0 &lt;= nums[i] &lt;= 100</li></ul>
