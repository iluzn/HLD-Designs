<h1>Find Pivot Index</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Prefix Sum</span></div></details>
<p>Given an array <code>nums</code>, return the leftmost <strong>pivot index</strong>: the index where the sum of all numbers strictly to the left equals the sum strictly to the right. Return <code>-1</code> if none exists.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,7,3,6,5,6]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3]
Output: -1</pre>
<h2>Example 3</h2><pre>Input:  nums = [2,1,-1]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^4</li><li>-1000 &lt;= nums[i] &lt;= 1000</li></ul>
