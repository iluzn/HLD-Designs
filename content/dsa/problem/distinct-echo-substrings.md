<h1>Distinct Echo Substrings</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Rolling Hash</span></div></details>
<p>Return the number of <strong>distinct</strong> non-empty substrings of <code>s</code> that can be written as the concatenation of some string with itself (i.e. equal to <code>a + a</code> for some string <code>a</code>).</p>
<h2>Example 1</h2><pre>Input:  s = "abcabcabc"
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  s = "leetcodeleetcode"
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  s = "aaaa"
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 16</li><li>s consists of lowercase English letters.</li></ul>
