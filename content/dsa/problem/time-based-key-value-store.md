<h1>Time Based Key-Value Store</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Design</span></div></details>
<p>Design <code>TimeMap</code> with <code>set(key, value, timestamp)</code> and <code>get(key, timestamp)</code>. get returns the value set for <code>key</code> with the largest timestamp <code>&lt;= timestamp</code>, or an empty string if none.</p>
<h2>Example 1</h2><pre>Input:  operations = ["TimeMap","set","get","get","set","get","get"], values = [[],["foo","bar",1],["foo",1],["foo",3],["foo","bar2",4],["foo",4],["foo",5]]
Output: [null,null,bar,bar,null,bar2,bar2]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= key.length, value.length &lt;= 100</li><li>set timestamps are strictly increasing per problem.</li><li>At most 2*10^5 calls.</li></ul>
