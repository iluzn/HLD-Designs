<h1>Reverse Bits</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Reverse the bits of a given 32-bit unsigned integer. Return the resulting unsigned integer represented as a (possibly large) number.</p><p>For example, the input <code>43261596</code> in binary is <code>00000010100101000001111010011100</code>. Reversing gives <code>00111001011110000010100101000000</code> which is <code>964176192</code>.</p>
<h2>Example 1</h2><pre>Input:  n = 43261596
Output: 964176192</pre>
<h2>Example 2</h2><pre>Input:  n = 4294967293
Output: 3221225471</pre>
<h2>Example 3</h2><pre>Input:  n = 0
Output: 0</pre>
<h2>Constraints</h2><ul><li>0 &lt;= n &lt;= 2^32 - 1</li></ul>
