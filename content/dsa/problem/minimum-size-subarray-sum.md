<h1>Minimum Size Subarray Sum</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sliding Window</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given an array of positive integers <code>nums</code> and a positive integer <code>target</code>, return the minimal length of a contiguous subarray whose sum is ≥ <code>target</code>. If there is none, return <code>0</code>.</p>
<h2>Example 1</h2><pre>Input:  target = 7, nums = [2,3,1,2,4,3]
Output: 2
Explanation: The subarray [4,3] has sum 7 and length 2.</pre>
<h2>Example 2</h2><pre>Input:  target = 11, nums = [1,1,1,1,1]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= 10^4</li><li>1 &lt;= target &lt;= 10^9</li></ul>
