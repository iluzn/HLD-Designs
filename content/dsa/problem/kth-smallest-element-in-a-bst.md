<h1>Kth Smallest Element in a BST</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BST</span></div></details>
<p>Given the <code>root</code> of a binary search tree and an integer <code>k</code>, return the <code>k</code>-th smallest value (1-indexed).</p>
<h2>Example 1</h2><pre>Input:  root = [3,1,4,null,2], k = 1
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  root = [5,3,6,2,4,null,null,1], k = 3
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= k &lt;= number of nodes &lt;= 10^4</li><li>0 &lt;= Node.val &lt;= 10^4</li></ul>
