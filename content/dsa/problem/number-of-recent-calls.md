<h1>Number of Recent Calls</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Queue</span><span class="lc-tag">Design</span><span class="lc-tag">Data Stream</span></div></details>
<p>Implement <code>RecentCounter</code> that counts recent requests within a time window. Implement <code>ping(t)</code> which records a request at time <code>t</code> (in milliseconds) and returns the number of requests that happened in the inclusive range <code>[t - 3000, t]</code>. Every call to <code>ping</code> uses a strictly larger <code>t</code> than the previous call.</p>
<h2>Example 1</h2><pre>Input:  operations = ["RecentCounter","ping","ping","ping","ping"], values = [[],[1],[100],[3001],[3002]]
Output: [null,1,2,3,3]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= t &lt;= 10^9</li><li>Each call to ping uses a strictly increasing t.</li><li>At most 10^4 calls to ping.</li></ul>
