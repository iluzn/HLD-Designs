<h1>Self Crossing</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Math</span><span class="lc-tag">Geometry</span></div></details>
<p>You start at the origin and move counter-clockwise: <code>nums[0]</code> north, <code>nums[1]</code> west, <code>nums[2]</code> south, <code>nums[3]</code> east, and so on. Given the distance array <code>nums</code>, return <code>true</code> if the path crosses itself, otherwise <code>false</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [2,1,1,2]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3,4]
Output: false</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,1,1,1]
Output: true</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 8</li><li>1 &lt;= nums[i] &lt;= 8</li></ul>
