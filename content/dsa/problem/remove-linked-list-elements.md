<h1>Remove Linked List Elements</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Recursion</span></div></details>
<p>Given the <code>head</code> of a linked list and an integer <code>k</code>, remove all nodes whose value equals <code>k</code> and return the new head. (LeetCode calls this parameter <code>val</code>.)</p>
<h2>Example 1</h2><pre>Input:  head = [1,2,6,3,4,5,6], k = 6
Output: [1,2,3,4,5]</pre>
<h2>Example 2</h2><pre>Input:  head = [7,7,7,7], k = 7
Output: []</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^4</li><li>0 &lt;= Node.val, k &lt;= 50</li></ul>
