<h1>Daily Temperatures</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Stack</span><span class="lc-tag">Monotonic Stack</span></div></details>
<p>Given an array <code>temperatures</code>, return an array <code>answer</code> where <code>answer[i]</code> is the number of days you must wait after day <code>i</code> for a warmer temperature. If none, put 0.</p>
<h2>Example 1</h2><pre>Input:  temperatures = [73,74,75,71,69,72,76,73]
Output: [1,1,4,2,1,1,0,0]</pre>
<h2>Example 2</h2><pre>Input:  temperatures = [30,40,50,60]
Output: [1,1,1,0]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= temperatures.length &lt;= 10^5</li><li>30 &lt;= temperatures[i] &lt;= 100</li></ul>
