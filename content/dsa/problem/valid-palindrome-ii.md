<h1>Valid Palindrome II</h1>

<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Greedy</span></div></details>
<p>Given a string <code>s</code>, return <code>true</code> if it can be a palindrome after deleting at most one character.</p>
<h2>Example 1</h2><pre>Input:  s = "aba"
Output: true</pre>
<h2>Example 2</h2><pre>Input:  s = "abca"
Output: true</pre>
<h2>Example 3</h2><pre>Input:  s = "abc"
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^5</li><li>s consists of lowercase English letters.</li></ul>
