<p>Given an array <code>nums</code> containing <code>n</code> distinct numbers in the range <code>[0, n]</code>, return the only number in the range that is missing.</p>
<h2>Example 1</h2><pre>Input:  nums = [3,0,1]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,1]
Output: 2</pre>
<h2>Constraints</h2><ul><li>n == nums.length</li><li>1 &lt;= n &lt;= 10^4</li><li>All numbers are distinct and in [0, n].</li></ul>
