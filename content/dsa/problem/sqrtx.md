<p>Given a non-negative integer <code>n</code>, return the integer square root of <code>n</code> (the floor of the true square root). Do not use built-in sqrt.</p>
<h2>Example 1</h2><pre>Input:  n = 4
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  n = 8
Output: 2
Explanation: floor(2.828...) = 2.</pre>
<h2>Constraints</h2><ul><li>0 &lt;= n &lt;= 2^31 - 1</li></ul>
