<h1>Min Cost Climbing Stairs</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an array <code>cost</code> where <code>cost[i]</code> is the cost of step <code>i</code>, you can climb 1 or 2 steps and start from index 0 or 1. Return the minimum cost to reach the top (just past the last step).</p>
<h2>Example 1</h2><pre>Input:  cost = [10,15,20]
Output: 15</pre>
<h2>Example 2</h2><pre>Input:  cost = [1,100,1,1,1,100,1,1,100,1]
Output: 6</pre>
<h2>Constraints</h2><ul><li>2 &lt;= cost.length &lt;= 1000</li><li>0 &lt;= cost[i] &lt;= 999</li></ul>
