<h1>Find Minimum in Rotated Sorted Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>A sorted array of unique integers was rotated at some pivot. Given the array <code>nums</code>, return the minimum element in O(log n).</p>
<h2>Example 1</h2><pre>Input:  nums = [3,4,5,1,2]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  nums = [4,5,6,7,0,1,2]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 5000</li><li>All integers are unique.</li><li>The array is a rotation of an ascending sort.</li></ul>
