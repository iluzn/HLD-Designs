<h1>Minimum Falling Path Sum II</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>n x n</code> integer <code>grid</code>, a falling path with non-zero shifts chooses exactly one element from each row such that no two chosen elements in adjacent rows are in the same column. Return the minimum sum of such a falling path.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,2,3],[4,5,6],[7,8,9]]
Output: 13</pre>
<h2>Example 2</h2><pre>Input:  grid = [[7]]
Output: 7</pre>
<h2>Constraints</h2><ul><li>n == grid.length == grid[i].length</li><li>1 &lt;= n &lt;= 200</li><li>-99 &lt;= grid[i][j] &lt;= 99</li></ul>
