<h1>Decode String</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Stack</span></div></details>
<p>Given an encoded string <code>s</code> following the rule <code>k[encoded]</code> meaning the bracketed part repeats k times, return the decoded string.</p>
<h2>Example 1</h2><pre>Input:  s = "3[a]2[bc]"
Output: "aaabcbc"</pre>
<h2>Example 2</h2><pre>Input:  s = "3[a2[c]]"
Output: "accaccacc"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 30</li><li>Digits are only for repeat counts.</li></ul>
