<h1>Rotate Image</h1>

<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Math</span><span class="lc-tag">Matrix</span></div></details>
<p>Given an <code>n x n</code> matrix, rotate it 90 degrees clockwise. LeetCode mutates in place and returns void; here return the rotated matrix.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[7,4,1],[8,5,2],[9,6,3]]</pre>
<h2>Example 2</h2><pre>Input:  grid = [[1,2],[3,4]]
Output: [[3,1],[4,2]]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 20</li><li>-1000 &lt;= matrix[i][j] &lt;= 1000</li></ul>
