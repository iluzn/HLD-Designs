<h1>Number of 1 Bits</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Given a non-negative integer <code>n</code>, return the number of set bits (1s) in its binary representation (its Hamming weight).</p>
<h2>Example 1</h2><pre>Input:  n = 11
Output: 3
Explanation: 11 = 1011 in binary.</pre>
<h2>Example 2</h2><pre>Input:  n = 128
Output: 1</pre>
<h2>Constraints</h2><ul><li>0 &lt;= n &lt;= 2^31 - 1</li></ul>
