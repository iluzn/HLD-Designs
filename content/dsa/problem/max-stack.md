<h1>Max Stack</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Stack</span><span class="lc-tag">Design</span><span class="lc-tag">Sorted Set</span></div></details>
<p>Design a stack that supports finding and removing the maximum element. Implement <code>MaxStack()</code> and: <code>push(x)</code>, <code>top()</code> (returns the top element), <code>pop()</code> (removes and returns the top element), <code>peekMax()</code> (returns the maximum element), and <code>popMax()</code> (removes and returns the maximum element; if there are ties, remove the one closest to the top).</p>
<h2>Example 1</h2><pre>Input:  operations = ["MaxStack","push","push","push","top","popMax","top","peekMax","pop","top"], values = [[],[5],[1],[5],[],[],[],[],[],[]]
Output: [null,null,null,null,5,5,1,5,1,5]</pre>
<h2>Constraints</h2><ul><li>-10^7 &lt;= x &lt;= 10^7</li><li>At most 10^4 calls.</li><li>top, pop, peekMax, popMax are only called on a non-empty stack.</li></ul>
