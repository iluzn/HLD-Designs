<h1>Coin Change II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>You are given coin denominations <code>nums</code> and a target amount <code>k</code>. Return the number of distinct combinations that make up the amount (order does not matter). Return 0 if impossible.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,5], k = 5
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  nums = [2], k = 3
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 300</li><li>1 &lt;= nums[i] &lt;= 5000</li><li>0 &lt;= k &lt;= 5000</li></ul>
