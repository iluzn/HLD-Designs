<h1>Running Sum of 1d Array</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Prefix Sum</span></div></details>
<p>Given an array <code>nums</code>, return its running sum where <code>runningSum[i] = sum(nums[0..i])</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,4]
Output: [1,3,6,10]</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,1,1,1,1]
Output: [1,2,3,4,5]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 1000</li><li>-10^6 &lt;= nums[i] &lt;= 10^6</li></ul>
