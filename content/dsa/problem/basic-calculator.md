<h1>Basic Calculator</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Math</span><span class="lc-tag">String</span><span class="lc-tag">Stack</span></div></details>
<p>Given a string <code>s</code> representing a valid expression, implement a basic calculator to evaluate it and return the result. The expression contains non-negative integers, <code>+</code>, <code>-</code>, <code>(</code>, <code>)</code>, spaces, and unary minus.</p>
<h2>Example 1</h2><pre>Input:  s = "1 + 1"
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  s = " 2-1 + 2 "
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  s = "(1+(4+5+2)-3)+(6+8)"
Output: 23</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 20</li><li>s consists of digits, '+', '-', '(', ')', and spaces.</li><li>s represents a valid expression.</li></ul>
