<h1>Find Champion II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span></div></details>
<p>There are <code>n</code> teams labeled 0..n-1 in a tournament. Each directed edge <code>[u, v]</code> means team <code>u</code> is stronger than team <code>v</code>. A team is the champion if no other team is stronger than it. Return the label of the unique champion, or <code>-1</code> if it is not unique.</p>
<h2>Example 1</h2><pre>Input:  n = 3, edges = [[0,1],[1,2]]
Output: 0</pre>
<h2>Example 2</h2><pre>Input:  n = 2, edges = [[0,1],[1,0]]
Output: -1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 100</li><li>edges[i] = [u, v] with u != v.</li></ul>
