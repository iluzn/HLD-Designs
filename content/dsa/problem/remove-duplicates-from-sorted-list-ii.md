<h1>Remove Duplicates from Sorted List II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given the <code>head</code> of a <strong>sorted</strong> linked list, delete <em>all</em> nodes that have duplicate values, leaving only values that appear exactly once. Return the sorted list.</p>
<h2>Example 1</h2><pre>Input:  head = [1,2,3,3,4,4,5]
Output: [1,2,5]</pre>
<h2>Example 2</h2><pre>Input:  head = [1,1,1,2,3]
Output: [2,3]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 300</li><li>-100 &lt;= Node.val &lt;= 100</li><li>The list is sorted in non-decreasing order.</li></ul>
