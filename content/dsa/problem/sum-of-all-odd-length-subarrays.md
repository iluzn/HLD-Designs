<p>Given an array of positive integers <code>arr</code>, return the sum of all elements over all contiguous subarrays of <strong>odd</strong> length.</p>
<h2>Example 1</h2><pre>Input:  arr = [1,4,2,5,3]
Output: 58</pre>
<h2>Example 2</h2><pre>Input:  arr = [1,2]
Output: 3</pre>
