<h1>Sum of All Odd Length Subarrays</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Prefix Sum</span><span class="lc-tag">Math</span></div></details>
<p>Given an array of positive integers <code>arr</code>, return the sum of all elements over all contiguous subarrays of <strong>odd</strong> length.</p>
<h2>Example 1</h2><pre>Input:  arr = [1,4,2,5,3]
Output: 58</pre>
<h2>Example 2</h2><pre>Input:  arr = [1,2]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= arr.length &lt;= 100</li><li>1 &lt;= arr[i] &lt;= 1000</li></ul>
