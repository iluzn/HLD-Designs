<h1>Monotonic Array</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span></div></details>
<p>An array is monotonic if it is entirely non-increasing or entirely non-decreasing. Given <code>nums</code>, return <code>true</code> if it is monotonic.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,2,3]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  nums = [6,5,4,4]
Output: true</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,3,2]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>-10^5 &lt;= nums[i] &lt;= 10^5</li></ul>
