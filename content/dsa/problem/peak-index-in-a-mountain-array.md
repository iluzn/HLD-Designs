<h1>Peak Index in a Mountain Array</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>An array <code>nums</code> is a <strong>mountain</strong>: it strictly increases up to a unique peak and then strictly decreases. Return the index of the peak element. Solve in O(log n) time.</p>
<h2>Example 1</h2><pre>Input:  nums = [0,1,0]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,2,1,0]
Output: 1</pre>
<h2>Example 3</h2><pre>Input:  nums = [0,10,5,2]
Output: 1</pre>
<h2>Constraints</h2><ul><li>3 &lt;= nums.length &lt;= 10^5</li><li>0 &lt;= nums[i] &lt;= 10^6</li><li>nums forms a mountain (unique peak).</li></ul>
