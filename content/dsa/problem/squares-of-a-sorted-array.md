<h1>Squares of a Sorted Array</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code> sorted in non-decreasing order, return an array of the squares of each number, also sorted in non-decreasing order.</p>
<h2>Example 1</h2><pre>Input:  nums = [-4,-1,0,3,10]
Output: [0,1,9,16,100]</pre>
<h2>Example 2</h2><pre>Input:  nums = [-7,-3,2,3,11]
Output: [4,9,9,49,121]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^4</li><li>-10^4 &lt;= nums[i] &lt;= 10^4</li><li>nums is sorted non-decreasing.</li></ul>
