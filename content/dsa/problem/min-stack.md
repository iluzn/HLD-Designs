<h1>Min Stack</h1>

<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Stack</span><span class="lc-tag">Design</span></div></details>
<p>Design a stack that supports push, pop, top, and retrieving the minimum element in constant time. Implement <code>MinStack</code> with <code>push(val)</code>, <code>pop()</code>, <code>top()</code>, and <code>getMin()</code>.</p>
<h2>Example 1</h2><pre>Input:  operations = ["MinStack","push","push","push","getMin","pop","top","getMin"], values = [[],[-2],[0],[-3],[],[],[],[]]
Output: [null,null,null,null,-3,null,0,-2]</pre>
<h2>Constraints</h2><ul><li>-2^31 &lt;= val &lt;= 2^31 - 1</li><li>pop, top, getMin are only called on non-empty stacks.</li><li>At most 3*10^4 calls.</li></ul>
