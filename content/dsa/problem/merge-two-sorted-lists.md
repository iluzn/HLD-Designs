<h1>Merge Two Sorted Lists</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Recursion</span></div></details>
<p>You are given the heads of two sorted linked lists <code>l1</code> and <code>l2</code>. Splice them into one sorted list and return its head.</p>
<h2>Example 1</h2><pre>Input:  l1 = [1,2,4], l2 = [1,3,4]
Output: [1,1,2,3,4,4]</pre>
<h2>Example 2</h2><pre>Input:  l1 = [], l2 = [0]
Output: [0]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= list length &lt;= 50</li><li>-100 &lt;= Node.val &lt;= 100</li><li>Both lists are sorted non-decreasing.</li></ul>
