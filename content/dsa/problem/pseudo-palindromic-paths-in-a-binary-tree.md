<h1>Pseudo-Palindromic Paths in a Binary Tree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Node values are digits <code>1-9</code>. A root-to-leaf path is <em>pseudo-palindromic</em> if at least one permutation of its node values is a palindrome. Return the number of pseudo-palindromic root-to-leaf paths.</p>
<h2>Example 1</h2><pre>Input:  root = [2,3,1,3,1,null,1]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  root = [2,1,1,1,3,null,null,null,null,null,1]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^5</li><li>1 &lt;= Node.val &lt;= 9</li></ul>
