<p>Given a string <code>s</code> with letters and <code>*</code>. Each <code>*</code> removes the closest non-star character to its left (and itself). Return the resulting string.</p>
<h2>Example 1</h2><pre>Input:  s = "leet**cod*e"
Output: "lecoe"</pre>
<h2>Example 2</h2><pre>Input:  s = "erase*****"
Output: ""</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^5</li><li>The operation is always possible.</li></ul>
