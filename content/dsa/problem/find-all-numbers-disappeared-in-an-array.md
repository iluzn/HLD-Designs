<h1>Find All Numbers Disappeared in an Array</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span></div></details>
<p>Given an array <code>nums</code> of <code>n</code> integers where <code>nums[i]</code> is in the range <code>[1, n]</code>, return an array (ascending) of all integers in <code>[1, n]</code> that do not appear in <code>nums</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [4,3,2,7,8,2,3,1]
Output: [5,6]</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,1]
Output: [2]</pre>
<h2>Constraints</h2><ul><li>n == nums.length</li><li>1 &lt;= n &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= n</li></ul>
