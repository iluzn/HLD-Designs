<h1>Binary Tree Tilt</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span></div></details>
<p>The <em>tilt</em> of a node is the absolute difference between the sum of all values in its left subtree and the sum of all values in its right subtree (an empty subtree sums to 0). Return the sum of every node’s tilt.</p>
<h2>Example 1</h2><pre>Input:  root = [1,2,3]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  root = [4,2,9,3,5,null,7]
Output: 15</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^4</li><li>-1000 &lt;= Node.val &lt;= 1000</li></ul>
