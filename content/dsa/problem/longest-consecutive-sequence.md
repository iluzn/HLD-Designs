<h1>Longest Consecutive Sequence</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Union Find</span></div></details>
<p>Given an unsorted array <code>nums</code>, return the length of the longest run of consecutive integers. You must do it in O(n) time.</p>
<h2>Example 1</h2><pre>Input:  nums = [100,4,200,1,3,2]
Output: 4
Explanation: The run [1,2,3,4] has length 4.</pre>
<h2>Example 2</h2><pre>Input:  nums = [0,3,7,2,5,8,4,6,0,1]
Output: 9</pre>
<h2>Constraints</h2><ul><li>0 &lt;= nums.length &lt;= 10^5</li><li>-10^9 &lt;= nums[i] &lt;= 10^9</li></ul>
