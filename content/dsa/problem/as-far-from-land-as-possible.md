<h1>As Far from Land as Possible</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">BFS</span><span class="lc-tag">Matrix</span></div></details>
<p>Given a grid where <code>1</code> is land and <code>0</code> is water, find the water cell whose Manhattan distance to the nearest land cell is maximized, and return that distance. If the grid has no water or no land, return <code>-1</code>.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,0,1],[0,0,0],[1,0,1]]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  grid = [[1,0,0],[0,0,0],[0,0,0]]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 100</li><li>grid[i][j] is 0 or 1.</li></ul>
