<h1>Array Partition</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code> of <code>2n</code> integers, group them into <code>n</code> pairs so that the sum of <code>min(pair)</code> over all pairs is maximized, and return that maximum sum.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,4,3,2]
Output: 4
Explanation: Pairs (1,2) and (3,4) give 1 + 3 = 4.</pre>
<h2>Example 2</h2><pre>Input:  nums = [6,2,6,5,1,2]
Output: 9</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 10^4</li><li>nums.length == 2n</li><li>-10^4 &lt;= nums[i] &lt;= 10^4</li></ul>
