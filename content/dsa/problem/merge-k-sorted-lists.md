<h2>Example 1</h2><pre>Input:  lists = [[1,4,5],[1,3,4],[2,6]]
Output: [1,1,2,3,4,4,5,6]</pre>
<h2>Example 2</h2><pre>Input:  lists = []
Output: []</pre>
<h2>Constraints</h2><ul><li>0 &lt;= k &lt;= 10^4</li><li>0 &lt;= list length &lt;= 500</li><li>Each list is sorted non-decreasing.</li></ul>
