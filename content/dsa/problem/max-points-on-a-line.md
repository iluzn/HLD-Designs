<h1>Max Points on a Line</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Math</span><span class="lc-tag">Geometry</span></div></details>
<p>You are given <code>grid</code>, where each row is a distinct point <code>[x, y]</code> on the plane. Return the maximum number of points that lie on the same straight line.</p>
<h2>Example 1</h2><pre>Input:  grid = [[1,1],[2,2],[3,3]]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  grid = [[1,1],[3,2],[5,3],[4,1],[2,3],[1,4]]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= grid.length &lt;= 300</li><li>grid[i].length == 2</li><li>all points are distinct</li></ul>
