<p>Given meeting intervals (each <code>[start, end]</code>), return the minimum number of conference rooms required.</p>
<h2>Example 1</h2><pre>Input:  grid = [[0,30],[5,10],[15,20]]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  grid = [[7,10],[2,4]]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= intervals.length &lt;= 10^4</li><li>intervals[i].length == 2</li></ul>
