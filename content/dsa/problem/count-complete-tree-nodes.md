<h1>Count Complete Tree Nodes</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given the <code>root</code> of a binary tree, return the total number of nodes.</p>
<h2>Example 1</h2><pre>Input:  root = [1,2,3,4,5,6]
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  root = []
Output: 0</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 5*10^4</li><li>-10^4 &lt;= Node.val &lt;= 10^4</li></ul>
