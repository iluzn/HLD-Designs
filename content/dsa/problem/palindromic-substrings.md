<h1>Palindromic Substrings</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Given a string <code>s</code>, return the number of <strong>palindromic substrings</strong> in it. Substrings at different start or end positions count separately even if identical.</p>
<h2>Example 1</h2><pre>Input:  s = "abc"
Output: 3
Explanation: The three single characters.</pre>
<h2>Example 2</h2><pre>Input:  s = "aaa"
Output: 6</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 1000</li><li>s consists of lowercase English letters.</li></ul>
