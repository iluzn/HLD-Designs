<h1>Remove Nodes From Linked List</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Stack</span><span class="lc-tag">Recursion</span><span class="lc-tag">Monotonic Stack</span></div></details>
<p>Given the <code>head</code> of a linked list, remove every node which has a node with a <strong>strictly greater</strong> value anywhere to its right. Return the head of the modified list.</p>
<h2>Example 1</h2><pre>Input:  head = [5,2,13,3,8]
Output: [13,8]</pre>
<h2>Example 2</h2><pre>Input:  head = [1,1,1,1]
Output: [1,1,1,1]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^5</li><li>1 &lt;= Node.val &lt;= 10^5</li></ul>
