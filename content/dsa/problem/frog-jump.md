<h1>Frog Jump</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Hash Table</span></div></details>
<p>A frog crosses a river by hopping on stones. The array <code>nums</code> gives the sorted positions of the stones (in units). The frog starts on the first stone (position 0) and its first jump must be exactly 1 unit. If the last jump was <code>k</code> units, the next jump must be <code>k-1</code>, <code>k</code>, or <code>k+1</code> units and must be strictly positive. Return <code>true</code> if the frog can land on the last stone.</p>
<h2>Example 1</h2><pre>Input:  stones = [0,1,3,5,6,8,12,17]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  stones = [0,1,2,3,4,8,9,11]
Output: false</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 2000</li><li>0 &lt;= nums[i] &lt;= 2^31 - 1</li><li>nums[0] == 0 and nums is strictly increasing.</li></ul>
