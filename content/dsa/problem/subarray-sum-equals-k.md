<h1>Subarray Sum Equals K</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Hash Table</span><span class="lc-tag">Prefix Sum</span></div></details>
<p>Given an integer array <code>nums</code> and an integer <code>k</code> (passed as <code>target</code>), return the total number of contiguous subarrays whose sum equals <code>k</code>.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,1,1], k = 2
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,3], k = 3
Output: 2</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 2*10^4</li><li>-1000 &lt;= nums[i] &lt;= 1000</li><li>nums may contain negatives (sliding window will not work).</li></ul>
