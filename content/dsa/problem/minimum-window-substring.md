<h1>Minimum Window Substring</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Hash Table</span><span class="lc-tag">String</span><span class="lc-tag">Sliding Window</span></div></details>
<p>Given two strings <code>s</code> and <code>t</code>, return the <strong>smallest substring</strong> of <code>s</code> that contains every character of <code>t</code> (including duplicates). If there is no such window, return the empty string <code>""</code>. When several windows share the minimum length, return the one with the smallest starting index.</p>
<h2>Example 1</h2><pre>Input:  s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"</pre>
<h2>Example 2</h2><pre>Input:  s = "a", t = "a"
Output: "a"</pre>
<h2>Example 3</h2><pre>Input:  s = "a", t = "aa"
Output: ""</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 16</li><li>1 &lt;= t.length &lt;= 6</li><li>s and t consist of lowercase and uppercase English letters.</li></ul>
