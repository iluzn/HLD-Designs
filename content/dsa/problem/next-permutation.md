<h1>Next Permutation</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Two Pointers</span></div></details>
<p>Rearrange <code>nums</code> into the lexicographically next greater permutation. If no greater permutation exists, rearrange to the lowest order (sorted ascending). Return the resulting array. Do it in-place with O(1) extra memory.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3]
Output: [1,3,2]</pre>
<h2>Example 2</h2><pre>Input:  nums = [3,2,1]
Output: [1,2,3]</pre>
<h2>Example 3</h2><pre>Input:  nums = [1,1,5]
Output: [1,5,1]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 100</li><li>0 &lt;= nums[i] &lt;= 100</li></ul>
