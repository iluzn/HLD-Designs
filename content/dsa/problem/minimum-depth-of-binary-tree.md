<h1>Minimum Depth of Binary Tree</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, return its minimum depth — the number of nodes along the shortest path from the root to the nearest <strong>leaf</strong> (a node with no children).</p>
<h2>Example 1</h2><pre>Input:  root = [3,9,20,null,null,15,7]
Output: 2</pre>
<h2>Example 2</h2><pre>Input:  root = [2,null,3,null,4,null,5,null,6]
Output: 5</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^5</li><li>-1000 &lt;= Node.val &lt;= 1000</li></ul>
