<h1>Count the Number of Complete Components</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Graph</span><span class="lc-tag">DFS</span><span class="lc-tag">BFS</span><span class="lc-tag">Union Find</span></div></details>
<p>Given <code>n</code> nodes labeled 0..n-1 and a list of undirected <code>edges</code>, return the number of <strong>complete connected components</strong>. A connected component is complete if every pair of its vertices is connected by a direct edge.</p>
<h2>Example 1</h2><pre>Input:  n = 6, edges = [[0,1],[0,2],[1,2],[3,4]]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  n = 6, edges = [[0,1],[0,2],[1,2],[3,4],[3,5]]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= n &lt;= 50</li><li>0 &lt;= edges.length &lt;= n*(n-1)/2</li><li>No repeated edges and no self-loops.</li></ul>
