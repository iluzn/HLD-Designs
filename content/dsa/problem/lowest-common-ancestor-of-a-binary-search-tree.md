<h1>Lowest Common Ancestor of a Binary Search Tree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BST</span></div></details>
<p>Given a binary search tree and two node values <code>p</code> and <code>q</code> present in it, return the value of their lowest common ancestor — the deepest node that has both as descendants (a node can be a descendant of itself).</p>
<h2>Example 1</h2><pre>Input:  root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4
Output: 2</pre>
<h2>Constraints</h2><ul><li>2 &lt;= number of nodes &lt;= 10^5</li><li>All node values are unique.</li><li>p and q exist in the BST.</li></ul>
