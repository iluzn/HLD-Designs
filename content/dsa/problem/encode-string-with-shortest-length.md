<h1>Encode String with Shortest Length</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given a string <code>s</code>, encode it so that its encoded length is as short as possible. The encoding rule is <code>k[encoded_string]</code>, where <code>encoded_string</code> is repeated exactly <code>k</code> times. If encoding does not make the string shorter, keep it as-is. When several encodings share the minimum length, return the <strong>lexicographically smallest</strong> one.</p>
<h2>Example 1</h2><pre>Input:  s = "aaa"
Output: "aaa"</pre>
<h2>Example 2</h2><pre>Input:  s = "aaaaa"
Output: "5[a]"</pre>
<h2>Example 3</h2><pre>Input:  s = "aaaaaaaaaa"
Output: "10[a]"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 12</li><li>s consists of lowercase English letters only.</li></ul>
