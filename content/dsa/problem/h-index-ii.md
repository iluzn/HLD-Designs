<h1>H-Index II</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span></div></details>
<p>Given an array <code>nums</code> of citation counts sorted in <strong>ascending</strong> order, return the researcher's <strong>h-index</strong>: the maximum value <code>h</code> such that at least <code>h</code> papers have at least <code>h</code> citations each. Solve in O(log n).</p>
<h2>Example 1</h2><pre>Input:  nums = [0,1,3,5,6]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [1,2,100]
Output: 2</pre>
<h2>Example 3</h2><pre>Input:  nums = [0]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>0 &lt;= nums[i] &lt;= 1000</li><li>nums is sorted in ascending order.</li></ul>
