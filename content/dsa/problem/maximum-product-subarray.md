<h1>Maximum Product Subarray</h1>

<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an integer array <code>nums</code>, find a contiguous non-empty subarray that has the largest <strong>product</strong>, and return the product.</p>
<h2>Example 1</h2><pre>Input:  nums = [2,3,-2,4]
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  nums = [-2,0,-1]
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 2*10^4</li><li>-10 &lt;= nums[i] &lt;= 10</li><li>Answer fits in a 32-bit integer.</li></ul>
