<h1>Bitwise AND of Numbers Range</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Bit Manipulation</span></div></details>
<p>Given two integers <code>a</code> and <code>b</code> with <code>a &lt;= b</code>, return the bitwise AND of all numbers in the inclusive range <code>[a, b]</code>.</p>
<h2>Example 1</h2><pre>Input:  a = 5, b = 7
Output: 4</pre>
<h2>Example 2</h2><pre>Input:  a = 0, b = 0
Output: 0</pre>
<h2>Example 3</h2><pre>Input:  a = 26, b = 30
Output: 24</pre>
<h2>Constraints</h2><ul><li>0 &lt;= a &lt;= b &lt;= 2^31 - 1</li></ul>
