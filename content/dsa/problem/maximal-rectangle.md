<h1>Maximal Rectangle</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span><span class="lc-tag">Stack</span><span class="lc-tag">Matrix</span></div></details>
<p>Given a rows-of-characters binary matrix <code>grid</code> filled with <code>'0'</code> and <code>'1'</code>, find the largest rectangle containing only <code>'1'</code>s and return its area.</p>
<h2>Example 1</h2><pre>Input:  matrix = ["10100","10111","11111","10010"]
Output: 6</pre>
<h2>Example 2</h2><pre>Input:  matrix = ["0"]
Output: 0</pre>
<h2>Example 3</h2><pre>Input:  matrix = ["1"]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= rows, cols &lt;= 200</li><li>grid[i][j] is '0' or '1'.</li></ul>
