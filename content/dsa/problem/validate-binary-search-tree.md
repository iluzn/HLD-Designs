<h1>Validate Binary Search Tree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">BST</span></div></details>
<p>Given the <code>root</code> of a binary tree, return <code>true</code> if it is a valid binary search tree: every node in the left subtree is strictly less, every node in the right subtree is strictly greater, and both subtrees are themselves valid BSTs.</p>
<h2>Example 1</h2><pre>Input:  root = [2,1,3]
Output: true</pre>
<h2>Example 2</h2><pre>Input:  root = [5,1,4,null,null,3,6]
Output: false</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 10^4</li><li>-2^31 &lt;= Node.val &lt;= 2^31 - 1</li></ul>
