<h1>Meeting Rooms</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Sorting</span><span class="lc-tag">Intervals</span></div></details>
<p>Given an array of meeting time <code>intervals</code> where <code>intervals[i] = [start, end]</code>, return <code>true</code> if a person could attend all meetings (no two overlap).</p>
<h2>Example 1</h2><pre>Input:  intervals = [[0,30],[5,10],[15,20]]
Output: false</pre>
<h2>Example 2</h2><pre>Input:  intervals = [[7,10],[2,4]]
Output: true</pre>
<h2>Constraints</h2><ul><li>0 &lt;= intervals.length &lt;= 10^4</li><li>intervals[i].length == 2</li><li>0 &lt;= start &lt; end</li></ul>
