<p>Given a string <code>s</code>, repeatedly remove two adjacent equal letters until no more can be removed. Return the final string.</p>
<h2>Example 1</h2><pre>Input:  s = "abbaca"
Output: "ca"</pre>
<h2>Example 2</h2><pre>Input:  s = "azxxzy"
Output: "ay"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 10^5</li><li>s consists of lowercase English letters.</li></ul>
