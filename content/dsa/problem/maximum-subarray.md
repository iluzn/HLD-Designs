<h1>Maximum Subarray</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Dynamic Programming</span></div></details>
<p>Given an integer array <code>nums</code>, find the contiguous subarray with the largest sum and return its sum.</p>
<h2>Example 1</h2><pre>Input:  nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: [4,-1,2,1] has sum 6.</pre>
<h2>Example 2</h2><pre>Input:  nums = [5,4,-1,7,8]
Output: 23</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>-10^4 &lt;= nums[i] &lt;= 10^4</li></ul>
