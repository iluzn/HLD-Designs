<h1>Count Nodes Equal to Average of Subtree</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span></div></details>
<p>For each node, consider the subtree rooted at it. Count the nodes whose value equals the <strong>average</strong> (rounded down) of all values in that subtree.</p>
<h2>Example 1</h2><pre>Input:  root = [4,8,5,0,1,null,6]
Output: 5</pre>
<h2>Example 2</h2><pre>Input:  root = [1]
Output: 1</pre>
<h2>Constraints</h2><ul><li>1 &lt;= number of nodes &lt;= 1000</li><li>0 &lt;= Node.val &lt;= 1000</li></ul>
