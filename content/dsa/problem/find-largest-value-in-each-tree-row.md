<h1>Find Largest Value in Each Tree Row</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">BFS</span></div></details>
<p>Given the <code>root</code> of a binary tree, return an array containing the <strong>largest</strong> value in each row (level), ordered from the top row to the bottom row.</p>
<h2>Example 1</h2><pre>Input:  root = [1,3,2,5,3,null,9]
Output: [1,3,9]</pre>
<h2>Example 2</h2><pre>Input:  root = [1,2,3]
Output: [1,3]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 10^4</li><li>-2^31 &lt;= Node.val &lt;= 2^31 - 1</li></ul>
