<p>Given a non-negative integer <code>n</code>, repeatedly add all its digits until the result has only one digit, and return it.</p>
<h2>Example 1</h2><pre>Input:  n = 38
Output: 2
Explanation: 3 + 8 = 11, 1 + 1 = 2.</pre>
<h2>Example 2</h2><pre>Input:  n = 0
Output: 0</pre>
