<h1>Binary Tree Postorder Traversal</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Tree</span><span class="lc-tag">DFS</span><span class="lc-tag">Stack</span></div></details>
<p>Given the <code>root</code> of a binary tree, return its <strong>postorder</strong> traversal (left, right, node) as a list of values.</p>
<h2>Example 1</h2><pre>Input:  root = [1,null,2,3]
Output: [3,2,1]</pre>
<h2>Example 2</h2><pre>Input:  root = [1,2,3,4,5]
Output: [4,5,2,3,1]</pre>
<h2>Constraints</h2><ul><li>0 &lt;= number of nodes &lt;= 100</li><li>-100 &lt;= Node.val &lt;= 100</li></ul>
