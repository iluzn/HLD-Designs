<h1>Maximum Ascending Subarray Sum</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span></div></details>
<p>Given an array <code>nums</code>, return the maximum possible sum of a strictly ascending contiguous subarray.</p>
<h2>Example 1</h2><pre>Input:  nums = [10,20,30,5,10,50]
Output: 65</pre>
<h2>Example 2</h2><pre>Input:  nums = [10,20,30,40,50]
Output: 150</pre>
<h2>Example 3</h2><pre>Input:  nums = [12,17,15,13,10,11,12]
Output: 33</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 100</li><li>1 &lt;= nums[i] &lt;= 100</li></ul>
