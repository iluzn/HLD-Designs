<h1>Maximum Count of Positive Integer and Negative Integer</h1>
<span class="lc-diff easy">Easy</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Counting</span></div></details>
<p>Given a sorted array <code>nums</code> in non-decreasing order, return the maximum between the number of strictly positive integers and the number of strictly negative integers. Zeros count toward neither. Aim for O(log n).</p>
<h2>Example 1</h2><pre>Input:  nums = [-2,-1,-1,1,2,3]
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [-3,-2,-1]
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  nums = [5,20,66,1314]
Output: 4</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 2000</li><li>-2000 &lt;= nums[i] &lt;= 2000</li><li>nums is sorted in non-decreasing order.</li></ul>
