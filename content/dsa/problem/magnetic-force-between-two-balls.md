<h1>Magnetic Force Between Two Balls</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Sorting</span></div></details>
<p>Given an array <code>nums</code> of distinct basket positions and an integer <code>target</code> (= m balls), place the balls into baskets so that the minimum distance between any two balls is as large as possible. Return that maximum possible minimum distance.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,4,7], target = 3
Output: 3</pre>
<h2>Example 2</h2><pre>Input:  nums = [5,4,3,2,1,1000000000], target = 2
Output: 999999999</pre>
<h2>Constraints</h2><ul><li>2 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= 10^9</li><li>All positions are distinct.</li><li>2 &lt;= target &lt;= nums.length</li></ul>
