<h1>Implement Queue using Stacks</h1>

<p>Implement a first-in-first-out (FIFO) queue using only two stacks. Implement <code>MyQueue</code> with <code>push(x)</code>, <code>pop()</code> (removes and returns the front element), <code>peek()</code> (returns the front element), and <code>empty()</code>.</p>
<h2>Example 1</h2><pre>Input:  operations = ["MyQueue","push","push","peek","pop","empty"], values = [[],[1],[2],[],[],[]]
Output: [null,null,null,1,1,false]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= x &lt;= 9</li><li>At most 100 calls to push, pop, peek, and empty.</li><li>pop and peek are only called on a non-empty queue.</li></ul>
