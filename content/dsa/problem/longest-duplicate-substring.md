<h1>Longest Duplicate Substring</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">String</span><span class="lc-tag">Binary Search</span><span class="lc-tag">Suffix Array</span><span class="lc-tag">Rolling Hash</span></div></details>
<p>Given a string <code>s</code>, consider all <strong>duplicated substrings</strong>: (contiguous) substrings that occur two or more times (the occurrences may overlap). Return the longest such substring; if there are several of the maximum length, return the <strong>lexicographically smallest</strong> one. If none exists, return <code>""</code>.</p>
<h2>Example 1</h2><pre>Input:  s = "banana"
Output: "ana"</pre>
<h2>Example 2</h2><pre>Input:  s = "abcd"
Output: ""</pre>
<h2>Example 3</h2><pre>Input:  s = "aaaa"
Output: "aaa"</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 16</li><li>s consists of lowercase English letters.</li></ul>
