<h1>Minimize Deviation in Array</h1>
<span class="lc-diff hard">Hard</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Array</span><span class="lc-tag">Greedy</span><span class="lc-tag">Heap</span></div></details>
<p>Given an array <code>nums</code> of positive integers, you may repeatedly apply two operations any number of times: if an element is even you may halve it, and if an element is odd you may double it. The deviation is the difference between the maximum and minimum elements. Return the minimum possible deviation.</p>
<h2>Example 1</h2><pre>Input:  nums = [1,2,3,4]
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  nums = [4,1,5,20,3]
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  nums = [2,10,8]
Output: 3</pre>
<h2>Constraints</h2><ul><li>1 &lt;= nums.length &lt;= 10^5</li><li>1 &lt;= nums[i] &lt;= 10^9</li></ul>
