<p>Given a parentheses string <code>s</code>, return the minimum number of <code>(</code> or <code>)</code> insertions needed to make it valid.</p>
<h2>Example 1</h2><pre>Input:  s = "())"
Output: 1</pre>
<h2>Example 2</h2><pre>Input:  s = "((("
Output: 3</pre>
<h2>Example 3</h2><pre>Input:  s = "()"
Output: 0</pre>
<h2>Constraints</h2><ul><li>1 &lt;= s.length &lt;= 1000</li><li>s consists of ( and ) only.</li></ul>
