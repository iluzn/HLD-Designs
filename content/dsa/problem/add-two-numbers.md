<h1>Add Two Numbers</h1>
<span class="lc-diff medium">Medium</span>
<details class="lc-topics"><summary>Topics</summary><div class="lc-tags"><span class="lc-tag">Linked List</span><span class="lc-tag">Math</span><span class="lc-tag">Recursion</span></div></details>
<p>You are given two non-empty linked lists representing two non-negative integers. The digits are stored in <strong>reverse order</strong>, one digit per node. Add the two numbers and return the sum as a linked list, also in reverse order.</p>
<h2>Example 1</h2><pre>Input:  l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807.</pre>
<h2>Example 2</h2><pre>Input:  l1 = [9,9,9], l2 = [1]
Output: [0,0,0,1]</pre>
<h2>Constraints</h2><ul><li>1 &lt;= list length &lt;= 100</li><li>0 &lt;= Node.val &lt;= 9</li><li>No leading zeros except the number 0 itself.</li></ul>
