# Code Judge

A LeetCode-style judge in your browser. Pick a problem, implement the function in **Python, Java, C++, or JavaScript**, run against the sample cases, then submit against the full hidden test set for an instant verdict — Accepted, Wrong Answer, Runtime Error, or Time Limit Exceeded. Your code is saved per problem in this browser.

## How it works

1. Open a problem — the description is on the left, the code editor on the right.
2. Pick your language. A function stub is provided; **you only implement the function** — input parsing is handled for you.
3. **Run** checks your code against the visible sample cases.
4. **Submit** runs every test case (including hidden ones) and returns a verdict.

Your code executes in a sandbox, so infinite loops are stopped by a time limit and crashes are reported as Runtime Errors.
